# Using React Magma

How to use React Magma components. The examples here use the <Link to="/api/button">Button</Link> component. More information about each of the components can be found in the Component API section.

## Basic Usage

To use a component, simply import the component and add it to your UI. The styles are already bundled with the component.
Here is a most basic example:

```.js
  import React from 'react'
  import { Button } from 'react-magma-dom'

  function Demo() {
    return <Button>Hello World</Button>
  }
```

## Importing React Magma Global Styles

Most of the styles for React Magma are bundled within their components. However, a `GlobalStyles` object is available, which provides a
very basic CSS reset plus the required fonts, body styles, and some generic styles for links and headings. It is recommended that you use this `GlobalStyles` component in any app that uses React Magma
to insure consistency across products.

To use the `GlobalStyles` object, simply import it from `react-magma-dom` and include it in your App.

The `GlobalStyles` component can be nested inside the <Link to="/api/theme">ThemeContext.Provivder</Link> to inherit styles from the theme.

```.js
  import React from 'react'
  import { GlobalStyles } from 'react-magma-dom'

  function App() {
    return <div>
      <GlobalStyles />
    </div>
  }
```

Some non-atomic components support styling of individual elements within the component via use of specific props.
For example, the <Link to="/api/checkbox">Checkbox</Link> supports the following properties:  `containerStyle`, `inputStyle`
and `labelStyle`.  In these cases, the `style` prop would have no effect because there is more than one element to target.

```.jsx
  <>
    <Checkbox
        checked
        containerStyle={{ 'border': '1px dotted purple'}}
        inputStyle={{ 'boxShadow': '3px 3px 3px rgba(0, 0 , 0, .6)'}}
        labelStyle={{ 'color': 'brown', 'fontStyle': 'italic'}}
        labelText='Custom Styles' />
  </>
```


## Using the Magma Theme in your Application

React Magma uses the Magma theme for styles, which includes fonts, colors and screen sizes for media queries. To remain consistent,
you can import `magma` and reference these values in your application's styles.

The complete Magma theme can be found in the <a href="http://stash.corp.web:7990/projects/FRONT/repos/react-magma/browse/packages/react-magma-dom/src/theme/magma.ts">Stash repository</a>.
For more information about colors, see the <Link to="/design/colors">Design Documentation</Link>.

```.js
  import React from 'react'
  import { magma } from 'react-magma-dom'
  import styled from '@emotion/styled';

  function Demo() {

    const StyledDiv = styled.div`
      background: ${magma.colors.neutral02};
      color: ${magma.colors.neutral08};
    `;

    return <StyledDiv>
      Hello world
    </StyledDiv>
  }
```


## Creating Custom Themes

Custom themes can be created by extending the Magma theme with your own values.  

```.js
  import { magma } from './magma';

  export const myCustomTheme = {
    ...magma,
    headingFont: 'Papyrus, sans-serif',

    colors: {
      ...magma.colors,

      primary: '#0000FF', 
      foundation01: '#FF0000',
    }
};
```


To apply the custom theme, use the <Link to="/api/theme/">ThemeContext.Provider</Link>, passing in your custom theme as the value property.
Any descendants of the context provider will receive the theme that is passed in.


```.js
  <ThemeContext.Provider value={myCustomTheme}>
    <div>....</div>
  </ThemeContext.Provider>
};
