# Icon Buttons

**Buttons are clickable items used to perform an action.**

Icon buttons have an icon included in them. Text is displayed in these buttons by default, but can be removed using the `iconOnly` property.
IconButton components accept all of the same props as Button components.

## Basic usage

```.jsx
    <p>
        <IconButton icon="bell" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="bell" variant="outline" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="bell" variant="link" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="cog2" color="secondary" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="cog2" color="secondary" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="cog2" color="secondary" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="check" color="success" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="check" color="success"variant="outline" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="check" color="success" variant="link" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="alert" color="danger" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="alert" color="danger" variant="outline" iconOnly={true}>Icon button</IconButton>
        <IconButton icon="alert" color="danger" variant="link" iconOnly={true}>Icon button</IconButton>
        <IconButton disabled icon="bell" iconOnly={true}>Icon button</IconButton>
    </p>
```

For icon only buttons you _MUST_ pass in an `ariaLabel` property to satisfy accessibility standards.

## Aria Label

```.jsx
    <p>
        <IconButton icon="bell" iconOnly={true} ariaLabel="Icon">Icon button</IconButton>
    </p>
```

## Text position

Use the optional prop `textPosition` to position the text of the button on either side of the icon.
The `textPosition` property takes the values `left` and `right`.

```.jsx
    <div>
      <p>
        <IconButton icon="bell" textPosition="left">Text Left</IconButton>
        <IconButton icon="info" color="secondary" textPosition="right">Text Right</IconButton>
      </p>
    </div>
```

## Sizes

Sizes for `IconButtons` include `small`, `medium`, and `large`, with `medium` being the default value.

```.jsx
    <div>
      <p>
          <IconButton icon="bell" size="small" iconOnly={true}>Icon button</IconButton>
          <IconButton icon="bell" iconOnly={true}>Icon button</IconButton>
          <IconButton icon="bell" size="large" iconOnly={true}>Icon button</IconButton>
      </p>
      <p>
          <IconButton icon="bell" size="small" textPosition="left">Small</IconButton>
          <IconButton icon="bell" textPosition="left">Medium</IconButton>
          <IconButton icon="bell" size="large" textPosition="left">Large</IconButton>
      </p>
    </div>
```

## Round vs square

Icon only buttons have `shape = round`, by default.

```.jsx
    <p>
      <IconButton icon="bell" iconOnly={true}>Icon button</IconButton>
      <IconButton icon="bell" shape="fill" iconOnly={true}>Icon button</IconButton>
      <IconButton icon="bell" shape="rightCap" iconOnly={true}>Icon button</IconButton>
      <IconButton icon="bell" shape="leftCap" iconOnly={true}>Icon button</IconButton>
    </p>
```

## Block Buttons

Passing the `block` property will set the button to display full-width.
Block buttons only work when the `icon` property is set to `false` or not defined.

```.jsx
    <p>
        <IconButton icon="bell" block textPosition="left">Block button</IconButton>
        <IconButton icon="bell" color="secondary" block textPosition="right">Block button</IconButton>
    </p>
```


## Using different elements
Passing in the `as` prop will allow a styed button to render as a different type of element, which can be useful
if you need to use an `a` tag or a `Link` component.

```.jsx
    <p>
        <IconButton
          ariaLabel='Search Google'
          as='a'
          href='http://google.com'
          icon="search2"
          textPosition='left'>Link Element</IconButton>
        <IconButton
          ariaLabel='Search Google'
          as='a'
          href='http://google.com'
          icon="search2"
          iconOnly />
    </p>
```

## Event Handling

The `handleClick` events will fire when the button is clicked, or when the enter or space bar key is pressed.
Events do not fire on disabled buttons.

```.jsx
  class Example extends React.Component {
      constructor(props) {
        super(props)
        this.state = {
          iconButtonCounter: 0,
          textIconButtonCounter: 0
        }
      }

      render() {
        return <div>
          <p><strong>Icon Counter: </strong> {this.state.iconButtonCounter} | <strong>Text Icon Counter: </strong> {this.state.textIconButtonCounter}</p>
          <IconButton
            icon="bell"
            iconOnly={true}
            handleClick={() => {
              this.setState(state => ({iconButtonCounter: state.iconButtonCounter + 1}))
            }}>Click me</IconButton>
          <IconButton
            icon="bell"
            textPosition="left"
            handleClick={() => {
              this.setState(state => ({textIconButtonCounter: state.textIconButtonCounter + 1}))
            }}>Click me</IconButton>
          <IconButton
            disabled
            icon="bell"
            iconOnly={true}
            handleClick={() => {
              this.setState(state => ({iconButtonCounter: state.iconButtonCounter + 1}))
            }}>Click me if you can</IconButton>
        </div>
      }
    }
```

| ariaLabel | string |  |
| as | any | |
| autoFocus | boolean | false |
| block | boolean | false |
| className | string | |
| children | node | |
| color | enum: 'primary' &#124; 'secondary' &#124; 'success' &#124; 'danger' | 'primary' |
| disabled | boolean | false |
| handleClick | func | | &#10003;
| href | string | |
| icon          | string | | &#10003; |
| iconOnly      | boolean | false |
| inverse | boolean | false |
| shape | enum: 'fill' &#124; 'leftCap' &#124; 'rightCap' &#124; 'round' | 'fill' |
