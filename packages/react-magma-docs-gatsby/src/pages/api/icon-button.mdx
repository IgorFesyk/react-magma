# Icon Buttons

**Buttons are clickable items used to perform an action.**

Icon buttons have an icon included in them.  Text is not displayed in these buttons by default, but can be using the textPostion property.
IconButton components accept all of the same props as Button components.

## Basic usage

```.jsx
    <p>
        <IconButton icon="bell">Icon button</IconButton>
        <IconButton icon="bell" variant="outline">Icon button</IconButton>
        <IconButton icon="bell" variant="link">Icon button</IconButton>
        <IconButton icon="cog2" color="secondary">Icon button</IconButton>
        <IconButton icon="cog2" color="secondary">Icon button</IconButton>
        <IconButton icon="cog2" color="secondary">Icon button</IconButton>
        <IconButton icon="check" color="success">Icon button</IconButton>
        <IconButton icon="check" color="success"variant="outline">Icon button</IconButton>
        <IconButton icon="check" color="success" variant="link">Icon button</IconButton>
        <IconButton icon="alert" color="danger">Icon button</IconButton>
        <IconButton icon="alert" color="danger" variant="outline">Icon button</IconButton>
        <IconButton icon="alert" color="danger" variant="link">Icon button</IconButton>
        <IconButton disabled icon="bell">Icon button</IconButton>
    </p>
```

## Text position

By default, an icon button will not display its children, and will instead use the child text as an `aria-label`.
However, if you use the optional prop `textPosition`, the icon button will display a regular button with both the icon and it's children.
The `textPosition` property takes the values `left` and `right`.

```.jsx
    <div>
      <p>
        <IconButton icon="bell" textPosition="left">Text Left</IconButton>
        <IconButton icon="info" color="secondary" textPosition="right">Text Right</IconButton>
      </p>
    </div>
```

## Sizes

Sizes for `IconButtons` include `small`, `medium`, and `large`, with `medium` being the default value.

```.jsx
    <p>
        <IconButton icon="bell" size="small">Icon button</IconButton>
        <IconButton icon="bell">Icon button</IconButton>
        <IconButton icon="bell" size="large">Icon button</IconButton>
    </p>
    <p>
        <IconButton icon="bell" size="small" textPosition="left">Small</IconButton>
        <IconButton icon="bell" textPosition="left">Medium</IconButton>
        <IconButton icon="bell" size="large" textPosition="left">Large</IconButton>
    </p>
```

## Round vs square

Icon only buttons have `shape = round`, by default.

```.jsx
    <p>
      <IconButton icon="bell">Icon button</IconButton>
      <IconButton icon="bell" shape="fill">Icon button</IconButton>
      <IconButton icon="bell" shape="rightCap">Icon button</IconButton>
      <IconButton icon="bell" shape="leftCap">Icon button</IconButton>
    </p>
```

##  Block Buttons
Passing the `block` property will set the button to display full-width.
Block buttons only work when the `textPosition` property is set to `left` or `right`.

```.jsx
    <p>
        <IconButton icon="bell" block textPosition="left">Block button</IconButton>
        <IconButton icon="bell" color="secondary" block textPosition="right">Block button</IconButton>
    </p>
```

## Event Handling

The `handleClick` events will fire when the button is clicked, or when the enter or space bar key is pressed.
Events do not fire on disabled buttons.

```.jsx
  class Example extends React.Component {
      constructor(props) {
        super(props)
        this.state = {
          iconButtonCounter: 0,
          textIconButtonCounter: 0
        }
      }

      render() {
        return <div>
          <p><strong>Icon Counter: </strong> {this.state.iconButtonCounter} | <strong>Text Icon Counter: </strong> {this.state.textIconButtonCounter}</p>
          <IconButton
            icon="bell"
            handleClick={() => {
              this.setState(state => ({iconButtonCounter: state.iconButtonCounter + 1}))
            }}>Click me</IconButton>
          <IconButton
            icon="bell"
            textPosition="left"
            handleClick={() => {
              this.setState(state => ({textIconButtonCounter: state.textIconButtonCounter + 1}))
            }}>Click me</IconButton>
          <IconButton
            disabled
            icon="bell"
            handleClick={() => {
              this.setState(state => ({iconButtonCounter: state.iconButtonCounter + 1}))
            }}>Click me if you can</IconButton>
        </div>
      }
    }
```

## Icon Button Props

| Prop                   | Variant  | Default | Required    |
| -----------------------|------ |---------|:-----------:|
| icon | string | | &#10003;
| handleClick | func | | &#10003;
| autoFocus | boolean | false |
| block | boolean | false |
| children | node | |
| color | enum: 'primary' &#124; 'secondary' &#124; 'success' &#124; 'danger' | 'primary' |
| disabled | boolean | false |
| id | string | |
| inverse | boolean | false |
| shape | enum: 'fill' &#124; 'leftCap' &#124; 'rightCap' &#124; 'round' | 'fill' |
| size | enum: 'large' &#124; 'medium' &#124; 'small' | 'medium' |
| textPosition | enum: 'left' &#124; 'right' | 'left' |
| textTransform | enum: 'uppercase' &#124; 'none' | 'uppercase' |
| variant | enum: 'solid' &#124; 'outline' &#124; 'link' | 'solid' |
