# Icon Buttons

**Buttons are clickable items used to perform an action.**

Icon buttons have an icon included in them.  Text is not displayed in these buttons by default, but can be using the textPostion property.
IconButton components accept all of the same props as Button components.

## Basic usage

```.jsx
    <p>
        <IconButton icon="bell" label="Icon button" />
        <IconButton icon="bell" type="outline" label="Icon button" />
        <IconButton icon="bell" type="link" label="Icon button" />
        <IconButton icon="cog2" color="secondary" label="Icon button" />
        <IconButton icon="cog2" color="secondary" type="outline" label="Icon button" />
        <IconButton icon="cog2" color="secondary" type="link" label="Icon button" />
        <IconButton icon="check" color="success" label="Icon button" />
        <IconButton icon="check" color="success"type="outline" label="Icon button" />
        <IconButton icon="check" color="success" type="link" label="Icon button" />
        <IconButton icon="alert" color="danger" label="Icon button" />
        <IconButton icon="alert" color="danger" type="outline" label="Icon button" />
        <IconButton icon="alert" color="danger" type="link" label="Icon button" />
        <IconButton disabled icon="bell" label="Icon button" />
    </p>
```

## Sizes

Sizes for `IconButtons` include `small`, `medium`, and `large`, with `medium` being the default value.

```.jsx
    <p>
        <IconButton icon="bell" size="small" label="Icon button" />
        <IconButton icon="bell" label="Icon button" />
        <IconButton icon="bell" size="large" label="Icon button" />
    </p>
```

## Round vs square

Icon only buttons have `shape = round`, by default.

```.jsx
    <p>
      <IconButton icon="bell" label="Icon button" />
      <IconButton icon="bell" shape="fill" label="Icon button" />
      <IconButton icon="bell" shape="rightCap" label="Icon button" />
      <IconButton icon="bell" shape="leftCap" label="Icon button" />
    </p>
```

## Text position

By default, an icon button will not display its label text, and will instead use the label text as an `aria-label`.
However, if you use the optional prop `textPosition`, the icon button will display a regular button with both the icon and the text.
The `textPosition` property takes the values `left` and `right`.  Using a button with more than one icon is not recommended,
but if you want to have a button with multiple icons, you can use the regular Button component and add the icons as the button's children.

```.jsx
    <p>
      <IconButton icon="bell" label="Text Left" textPosition="left" />
      <IconButton icon="info" color="secondary" label="Text Right" textPosition="right" />
    </p>
```

## Event Handling

The `handleClick` events will fire when the button is clicked, or when the enter or space bar key is pressed.
Events do not fire on disabled buttons.

```.jsx
  class Example extends React.Component {
      constructor(props) {
        super(props)
        this.state = {
          iconButtonCounter: 0,
          textIconButtonCounter: 0
        }
      }

      render() {
        return <div>
          <p><strong>Icon Counter: </strong> {this.state.iconButtonCounter} | <strong>Text Icon Counter: </strong> {this.state.textIconButtonCounter}</p>
          <IconButton
            icon="bell"
            label="Bell Icon"
            handleClick={() => {
              this.setState(state => ({iconButtonCounter: state.iconButtonCounter + 1}))
            }}
          />
          <IconButton
            icon="bell"
            label="Click Me"
            textPosition="left"
            handleClick={() => {
              this.setState(state => ({textIconButtonCounter: state.textIconButtonCounter + 1}))
            }}
          />
          <IconButton
            disabled
            icon="bell"
            label="Bell Icon"
            handleClick={() => {
              this.setState(state => ({iconButtonCounter: state.iconButtonCounter + 1}))
            }}
          />
        </div>
      }
    }
```
