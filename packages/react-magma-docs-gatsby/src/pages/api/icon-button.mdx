# Icon Buttons

**Icon Buttons use the Button component to render. It is not a separate component, just different props.**

**Buttons are clickable items used to perform an action.**

Icon buttons have an icon included in them. Text is displayed in these buttons by default, but can be removed using the `iconOnly` property.
IconButton components accept all of the same props as Button components.

## Basic usage

```.jsx
    <p>
        <Button ariaLabel="Icon" icon={<BellIcon />} />
        <Button ariaLabel="Icon" variant="outline" icon={<BellIcon />} />
        <Button ariaLabel="Icon" variant="link" icon={<BellIcon />} />
        <Button ariaLabel="Icon" color="secondary" icon={<Cog2Icon />} />
        <Button ariaLabel="Icon" color="secondary" variant="outline" icon={<Cog2Icon />} />
        <Button ariaLabel="Icon" color="secondary" variant="link" icon={<Cog2Icon />} />
        <Button ariaLabel="Icon" color="success" icon={<CheckIcon />} />
        <Button ariaLabel="Icon" color="success"variant="outline" icon={<CheckIcon />} />
        <Button ariaLabel="Icon" color="success" variant="link" icon={<CheckIcon />} />
        <Button ariaLabel="Icon" color="danger" icon={<AlertIcon />} />
        <Button ariaLabel="Icon" color="danger" variant="outline" icon={<AlertIcon />} />
        <Button ariaLabel="Icon" color="danger" variant="link" icon={<AlertIcon />} />
        <Button ariaLabel="Icon" disabled icon={<BellIcon />} />
    </p>
```

## Assign Icon to Variable

If you find JSX being passed in as a prop directly ugly, you can assign an icon to a variable and pass that in as the prop.

```.jsx
  class Example extends React.Component {
      render() {
        const icon = <BellIcon />;

        return <Button ariaLabel="Icon" icon={icon} />
      }
    }
```

For icon only buttons you _MUST_ pass in an `ariaLabel` property to satisfy accessibility standards.

## Aria Label

```.jsx
    <p>
        <Button ariaLabel="Icon" icon={<BellIcon />} />
    </p>
```

## Icon position

Use the optional prop `iconPosition` to position the icon of the button on either side of the text.
The `iconPosition` property takes the values `left` and `right`.

```.jsx
    <div>
      <p>
        <Button icon={<BellIcon />}>Icon Left</Button>
        <Button color="secondary" icon={<InfoIcon />} iconPosition="right">Icon Right</Button>
      </p>
    </div>
```

## Sizes

Sizes for `IconButtons` include `small`, `medium`, and `large`, with `medium` being the default value.

```.jsx
    <div>
      <p>
          <Button icon={<BellIcon />} size="small" />
          <Button icon={<BellIcon />} />
          <Button icon={<BellIcon />} size="large" />
      </p>
      <p>
          <Button icon={<BellIcon />} size="small" iconPosition="left">Small</Button>
          <Button icon={<BellIcon />} iconPosition="left">Medium</Button>
          <Button icon={<BellIcon />} size="large" iconPosition="left">Large</Button>
      </p>
    </div>
```

## Round vs square

Icon only buttons have `shape = round`, by default.

```.jsx
    <p>
      <Button icon={<BellIcon />} />
      <Button icon={<BellIcon />} shape="fill" />
      <Button icon={<BellIcon />} shape="rightCap" />
      <Button icon={<BellIcon />} shape="leftCap" />
    </p>
```

## Block Buttons

Passing the `block` property will set the button to display full-width.
Block buttons only work when the `icon` property is set to `false` or not defined.

```.jsx
    <p>
        <Button icon={<BellIcon />} block iconPosition="left">Block button</Button>
        <Button icon={<BellIcon />} color="secondary" block iconPosition="right">Block button</Button>
    </p>
```

## Using different elements

Passing in the `as` prop will allow a styed button to render as a different type of element, which can be useful
if you need to use an `a` tag or a `Link` component.

```.jsx
    <p>
        <Button
          ariaLabel='Search Google'
          as='a'
          href='http://google.com'
          icon={<Search2Icon />}
          iconPosition='left'>Link Element</Button>
        <Button
          ariaLabel='Search Google'
          as='a'
          href='http://google.com'
          icon={<Search2Icon />} />
    </p>
```

## Event Handling

The `onClick` event will fire when the button is clicked, or when the enter or space bar key is pressed.
Events do not fire on disabled buttons.

```.jsx
  class Example extends React.Component {
      constructor(props) {
        super(props)
        this.state = {
          iconButtonCounter: 0,
          textIconButtonCounter: 0
        }
      }

      render() {
        return <div>
          <p><strong>Icon Counter: </strong> {this.state.iconButtonCounter} | <strong>Text Icon Counter: </strong> {this.state.textIconButtonCounter}</p>
          <Button
            icon={<BellIcon />}
            onClick={() => {
              this.setState(state => ({iconButtonCounter: state.iconButtonCounter + 1}))
            }}>Click me</Button>
          <Button
            icon={<BellIcon />}
            textPosition="left"
            onClick={() => {
              this.setState(state => ({textIconButtonCounter: state.textIconButtonCounter + 1}))
            }}>Click me</Button>
          <Button
            disabled
            icon={<BellIcon />}
            onClick={() => {
              this.setState(state => ({iconButtonCounter: state.iconButtonCounter + 1}))
            }}>Click me if you can</Button>
        </div>
      }
    }
```

## Testing

Passing in the `testId` prop will add the `data-testid` attribute to the button element for easier querying in tests.

```html

    class="icon"
    height="18"
    width="18"
    fill="currentColor"
    xmlns="http://www.w3.org/2000/svg"
    viewBox="0 0 21.17 23.56"
    aria-labelledby="5c3f5f70-81b2-4793-90ad-f69fca370f6a"
  >
    <title id="5c3f5f70-81b2-4793-90ad-f69fca370f6a"></title>

      d="M21.9,10.31c0-4.8-5.34-6.49-5.34-6.49V2.22h-2v1.6S9.22,5.51,9.22,10.31,7,20.69,5,20.69v1.9h7.38a3.2,3.2,0,0,0,6.39,0h7.39v-1.9c-2,0-4.24-5.59-4.24-10.38Z"
      transform="translate(-4.98 -2.22)"
    ></path>
  </svg>
</button>
```

## Icon Button Props

| ------------- | ------------------------------------------------------------------- | ----------- | :------: |
| ariaLabel     | string                                                              |             |
| as            | any                                                                 |             |
| autoFocus     | boolean                                                             | false       |
| block         | boolean                                                             | false       |
| className     | string                                                              |             |
| children      | node                                                                |             |
| color         | enum: 'primary' &#124; 'secondary' &#124; 'success' &#124; 'danger' | 'primary'   |
| disabled      | boolean                                                             | false       |
| onClick       | func                                                                |             | &#10003; |
| icon          | Icon ReactElement                                                   |             |
| id            | string                                                              |             |          |
| inverse       | boolean                                                             | false       |
| shape         | enum: 'fill' &#124; 'leftCap' &#124; 'rightCap' &#124; 'round'      | 'fill'      |
| size          | enum: 'large' &#124; 'medium' &#124; 'small'                        | 'medium'    |
| style         | React.CSSProperties                                                 |             |
| iconPosition  | enum: 'left' &#124; 'right'                                         |             |
| testId        | string                                                              |             |          |
| textTransform | enum: 'uppercase' &#124; 'none'                                     | 'uppercase' |
| to            | string                                                              |             |
