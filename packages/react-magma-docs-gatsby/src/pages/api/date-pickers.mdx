# Date Pickers

## Basic Usage

```.jsx
  <div>
    <DatePicker labelText="Date" />
  </div>
```

## Click Events

```.jsx
  class Example extends React.Component {
    constructor(props) {
      super(props)
      this.state = {
        chosenDate: 0
      }
    }

    render() {
      const { chosenDate } = this.state;
      return <div>
      <p><strong>Chosen Date: </strong> {chosenDate && <span>{this.state.chosenDate.format('dddd, LL')}</span>}</p>
      <DatePicker
        onDayClick={(chosenDate) => {
          this.setState(state => ({chosenDate}))
        }} />
      </div>
    }
  }
```
