# Toggles

## Basic Usage

```.jsx
  <div>
    <Toggle
        id='Toggle0'
        labelText='Toggle label' />
  </div>
```

## Colors
By default, the toggles use the theme's success color.  But any other color may be passed in.

```.jsx
  <div>
  <Toggle
      id='colorToggle'
      labelText='Default Color'
      value={false} />
    <Toggle
        color={magma.colors.primary}
        id='successColorToggle'
        labelText='Primary Color' />
    <Toggle
        color={magma.colors.danger}
        id='dangerColorToggle'
        labelText='Danger Color' />
    <Toggle
        color={magma.colors.pop02}
        id='popColorToggle'
        labelText='Pop Color' />
  </div>
```

## Disabled

```.jsx
  <div>
    <Toggle
        disabled
        id='disabledToggle'
        labelText='Disabled Toggle' />
    </div>
```

## Visually Hidden Label
Adding the `textVisuallyHidden` prop will make the label text available to screen readers, but will hide it visually.

```.jsx
  <div>
    <Toggle
        id='visuallyHiddenToggle'
        labelText='Label is hidden'
        textVisuallyHidden />
    </div>
```
