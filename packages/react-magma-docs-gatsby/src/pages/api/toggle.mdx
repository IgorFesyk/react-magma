## Basic Usage

The label/toggle association will be handled via the passed in `id` if one is supplied or via an auto-generated `id` if not supplied.

```.jsx
  <div>
    <Toggle
        id='Toggle0'
        labelText='Toggle label' />
    <Toggle
        checked
        id='Toggle1'
        labelText='Toggle label on' />
  </div>
```

## Disabled

```.jsx
  <div>
    <Toggle
        disabled
        id='disabledToggle'
        labelText='Disabled Toggle' />
    <Toggle
        checked
        disabled
        id='disabledToggleOn'
        labelText='Disabled Toggle On' />
    </div>
```

## Text position

Use the optional prop `textPosition` to position the label text on either side of the toggle switch.
The `textPosition` property takes the values `left` and `right`. The default value is `left`.

```.jsx
  <div>
    <Toggle
        id='ToggleTextLeft'
        labelText='Text left' />
    <Toggle
        id='ToggleTextRight'
        labelText='Text right'
        textPosition='right' />
    </div>
```

## Visually Hidden Label

Adding the `textVisuallyHidden` prop will make the label text available to screen readers, but will hide it visually.

```.jsx
  <div>
    <Toggle
        id='visuallyHiddenToggle'
        labelText='Label is hidden'
        textVisuallyHidden />
    </div>
```

## Custom Styles

Custom styles can be passed into both the Toggle component. The `style` property will apply to the container.
Additional `labelStyle`, `trackStyle` and `thumbStyle` properties are available to style the respective elements.
Please use discression when adding custom styles.

```.jsx
  <div>
    <Toggle
        id='custom1'
        labelStyle={{ 'fontStyle': 'italic', 'flexGrow': '1'}}
        labelText='Custom Styles'
        style={{ 'border': '1px dotted purple', 'width': '100%'}}
        thumbStyle={{ 'background': 'yellow', 'border' : '1px solid orange', 'boxShadow': '0 0 0' }}
        trackStyle={{ 'borderRadius': '5px', 'height': '18px'}} />
  </div>
```

## Testing

```html

  <label for="Toggle0">
    <span>Toggle label</span>
```

| checked            | boolean                     |         |
| disabled           | boolean                     |         |

| onChange           | function                    |         |
| onFocus            | function                    |         |
| id                 | string                      |         |          |
| labelStyle         | React.CSSProperties         |         |
| labelText          | string                      |         | &#10003; |
| required           | function                    |         |
| style              | React.CSSProperties         |         |

| testId             | string                      |         |
| thumbStyle         | React.CSSProperties         |         |
| trackStyle         | React.CSSProperties         |         |
