# Tooltip

## Bottom

```.jsx
    <Tooltip position='bottom' content={<>Tooltip content</>}>
        Trigger content
    </Tooltip>
```

## Left

```.jsx
    <Tooltip position='left' content="Tooltip content">
        Trigger content
    </Tooltip>
```

## Right

```.jsx
    <Tooltip position='right' content={<>Tooltip content</>}>
        Trigger content
    </Tooltip>
```

## Top

```.jsx
    <Tooltip position='top' content="Tooltip content">
        Trigger content
    </Tooltip>
```

## Tooltip Props

| Prop     | Type                                                     | Default | Required |
| -------- | -------------------------------------------------------- | ------- | :------: |
| children | node                                                     |         | &#10003; |
| content  | node                                                     |         | &#10003; |
| position | enum: 'bottom' &#124; 'left' &#124; 'right' &#124; 'top' | 'left'  |          |
