# Checkboxes

## Basic Usage

```.jsx
  <div>
    <Checkbox
        checked
        id='checkbox0'
        labelText='Checkbox Label' />

    <Checkbox
        id='checkbox1'
        labelText='Checkbox Label' />

    <Checkbox
        id='checkbox2'
        labelText='Checkbox Label is really long and can wrap to multiple lines lorem ipsum dolar sit amet is really long and can wrap to multiple lines' />
    </div>
```

## Colors
By default, the checkboxes use the theme's primary color.  But any other color may be passed in.

```.jsx
  <div>
    <Checkbox
        checked
        id='colorCheckbox'
        labelText='Default Color' />
    <Checkbox
        checked
        color={magma.colors.success01}
        id='primaryColorCheckbox'
        labelText='Success Color' />
    <Checkbox
        checked
        color={magma.colors.danger}
        id='dangerColorCheckbox'
        labelText='Danger Color' />
    <Checkbox
        checked
        color={magma.colors.pop02}
        id='popColorCheckbox'
        labelText='Pop Color' />
    </div>
```

## Disabled

```.jsx
  <div>
    <Checkbox
        disabled
        id='disabledCheckbox'
        labelText='Disabled Checkbox' />
    </div>
```

## Indeterminate

```.jsx
  <div>
    <Checkbox
        id='indeterminateCheckbox'
        indeterminate
        labelText='Indeterminate Checkbox' />
    <Checkbox
        color={magma.colors.success01}
        id='indeterminateSuccessCheckbox'
        indeterminate
        labelText='Indeterminate Checkbox' />
    </div>
```

## Inverse colors
Inverse is an optional boolean prop, that may be passed in when checkboxes are to be displayed on a dark background.

```.jsx
  <div style={{ background : '#00263E', padding: '10px' }}>
    <Checkbox
        checked
        id='inverseCheckbox'
        inverse
      labelText='Inverse' />
    <Checkbox
        checked
        color={magma.colors.success01}
        id='successInverseCheckbox'
        inverse
        labelText='Success Inverse' />
    <Checkbox
        checked
        color={magma.colors.danger}
        id='dangerInverseCheckbox'
        inverse
        labelText='Danger Inverse' />
    <Checkbox
        checked
        color={magma.colors.pop02}
        id='popInverseCheckbox'
        inverse
        labelText='Pop Inverse' />
      <Checkbox
          disabled
          id='disabledInverseCheckbox'
          inverse
          labelText='Disabled Inverse' />
      <Checkbox
          id='indeterminateInverseCheckbox'
          indeterminate
          inverse
          labelText='Indeterminate Checkbox' />
    </div>
```

## Visually Hidden Label
Adding the `textVisuallyHidden` prop will make the label text available to screen readers, but will hide it visually.

```.jsx
  <div>
    <Checkbox
        id='visuallyHiddenCheckbox'
        labelText='Disabled Checkbox'
        textVisuallyHidden />
    </div>
```


## Custom Styles
Custom styles can be passed into both the Checkbox component.  The `style` property will apply to the container.
Additional `labelStyle` and `inputStyle` properties are available to style the respective elements. Please use discression when adding custom styles.

```.jsx
  <div>
    <Checkbox
        checked
        id='custom1'
        inputStyle={{ 'boxShadow': '3px 3px 3px rgba(0, 0 , 0, .6)'}}
        labelStyle={{ 'color': 'red', 'fontStyle': 'italic'}}
        labelText='Custom Styles'
        style={{ 'border': '1px dotted purple'}} />

    <Checkbox
        checked
        id='custom2'
        labelText='No Custom Styles' />
  </div>
```
