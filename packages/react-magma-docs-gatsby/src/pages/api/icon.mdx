# Icons

## Basic Usage

```.jsx
  <p>
    <span style={{"padding": "20px", "text-align": "center", "margin": "10px", "min-width": "200px"}}>
      <Icon id="infoBasic" title="Info Available" type="info" />
    </span>
  </p>
```

## Sizes

```.jsx
  <p>
    <span style={{"padding": "20px", "text-align": "center", "margin": "10px", "min-width": "200px"}}>
      <Icon id="smallSize" size={20} title="Small Size" type="angle-down" />
    </span>
    <span style={{"padding": "20px", "text-align": "center", "margin": "10px", "min-width": "200px"}}>
      <Icon id="largeSize" size={50} title="Large Size" type="angle-down" />
    </span>
  </p>
```

## Color

```.jsx
  <p>
    <span style={{"padding": "20px", "text-align": "center", "margin": "10px", "min-width": "200px"}}>
      <Icon id="noColor" title="No Color" type="angle-down" />
    </span>
    <span style={{"padding": "20px", "text-align": "center", "margin": "10px", "min-width": "200px"}}>
      <Icon id="someColor" color="green" title="Some Color" type="angle-down" />
    </span>
  </p>
```

## Available Icons

```.jsx
    <div style={{"display": "flex", "flex-wrap": "wrap", "justify-content": "space-between"}}>{Object.keys(ICONS).map((key) => {
      const id = key + 'Available';
      const title = key + ' Title';

      return <span style={{"padding": "20px", "text-align": "center", "margin": "10px", "min-width": "150px"}}>
        <Icon id={id} title={title} type={key} /><br />
        {key}
      </span>
    })}</div>
```
