# Buttons

**Buttons are clickable items used to perform an action.**

Buttons should be used in situations where users might need to:

* Submit a form.
* Begin a new task.
* Trigger a new UI element to appear on the page.
* Specify a new or next step in a process.

## Icon Buttons

```.jsx
    <p>
        <IconButton text="Icon button" />
        <IconButton type="outline" text="Icon button" />
        <IconButton type="link" text="Icon button" />
        <IconButton color="secondary" text="Icon button" />
        <IconButton color="success" text="Icon button" />
        <IconButton color="danger" text="Icon button" />
        <IconButton color="danger" type="outline" text="Icon button" />
    </p>
```

## Event Handling

```.jsx
  class Example extends React.Component {
      constructor(props) {
        super(props)
        this.state = {
          counter: 0
        }
      }

      render() {
        return <div>
        <p><strong>Counter: </strong> <span id="counter">{this.state.counter}</span></p>
        <Button
          handleClick={() => {
            this.setState(state => ({counter: state.counter + 1}))
          }}>Click me</Button>
        </div>
      }
    }
```
