# Buttons

**Buttons are clickable items used to perform an action.**

Buttons should be used in situations where users might need to:

* Submit a form.
* Begin a new task.
* Trigger a new UI element to appear on the page.
* Specify a new or next step in a process.

## Sizes

```.jsx
    <p>
        <Button type="primary" size="small" text="Small" />
        <Button type="primary" text="Medium" />
        <Button type="primary" size="large" text="Large" />
    </p>
```

## Colors

```.jsx
    <p>
        <Button color="primary" text="Primary" />
        <Button color="secondary" text="Secondary" />
        <Button color="success" text="Success" />
        <Button color="danger" text="Danger" />
    </p>
```

## Types

```.jsx
    <p>
        <Button color="primary" type="Solid" text="Solid" />
        <Button color="primary" type="Outline" text="Outline" />
        <Button color="primary" type="Link" text="Link" />
    </p>
```

## Shapes

```.jsx
    <p>
        <Button color="primary" text="Fill (default)" />
        <Button color="primary" shape="leftCap" text="Left Cap" />
        <Button color="primary" shape="rightCap" text="Right Cap" />
    </p>
```

## Inverse

```.jsx
    <p>
        <Button color="primary" inverse text="Primary inverse" />
        <Button color="secondary" inverse text="Secondary inverse" />
        <Button color="success" inverse text="Success inverse" />
        <Button color="danger" inverse text="Danger inverse" />
    </p>
```

## All Caps

```.jsx
    <p>
        <Button color="primary" allCaps text="All caps" />
        <Button color="primary" allCaps={false} text="All caps false" />
    </p>
```

## Disabled

```.jsx
    <p>
        <Button disabled type="primary" text="Disabled" />
    </p>
```

## Event Handling

```.jsx
  class Example extends React.Component {
      constructor(props) {
        super(props)
        this.state = {
          counter: 0
        }
      }

      render() {
        return <div>
        <p><strong>Counter: </strong> <span id="counter">{this.state.counter}</span></p>
        <Button
          handleClick={() => {
            this.setState(state => ({counter: state.counter + 1}))
          }}
          text="Click me" />
        </div>
      }
    }
```
