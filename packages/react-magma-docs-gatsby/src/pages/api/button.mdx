# Buttons

## Basic Usage

```.jsx
  <Button text="Hello" />
```

## Event Handling

```.jsx
  class Example extends React.Component {
      constructor(props) {
        super(props)
        this.state = {
          counter: 0
        }
      }

      render() {
        return <div>
        <p><strong>Counter: </strong> <span id="counter">{this.state.counter}</span></p>
        <Button
          handleClick={() => {
            this.setState(state => ({counter: state.counter + 1}))
          }}
          text="Click me" />
        </div>
      }
    }
```
