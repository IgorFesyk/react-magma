# Textarea

## Basic Usage

```.jsx
    <Textarea labelText="Textarea" value="Some text" />
```

## Default Value

```.jsx
    <Textarea labelText="Textarea" defaultValue="Some text" />
```

## Error Message

```.jsx
    <Textarea errorMessage="Please fix this error" labelText="Textarea with Error" />
```

## Helper Message

```.jsx
    <Textarea helperMessage="Helper text goes here" labelText="Textarea with Helper" />
```

## Disabled

```.jsx
    <Textarea disabled labelText="Disabled Textarea" />
```

## Inverse

```.jsx
    <div style={{ background : '#00263E', padding: '10px' }}>
      <Textarea isInverse labelText="Inverse Textarea" />
      <Textarea errorMessage="Please fix this error" isInverse labelText="Inverse Textarea with Error" />
    </div>
```
