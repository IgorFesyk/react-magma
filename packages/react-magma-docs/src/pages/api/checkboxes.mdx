## Basic Usage

Checkboxes are form elements used to select one or multiple items from a list. The label/checkbox association will be handled via the passed in `id` if one is supplied or via an auto-generated `id` if not supplied.

See also: <Link to="/api/radio">Radio Buttons</Link>, <Link to="/api/toggle">Toggle Switches</Link>, and the <Link to="/design/selection-controls">Design Guidelines</Link> for selection controls in general.

```.jsx
  <div>
    <Checkbox
        checked
        id='checkbox0'
        labelText='Checkbox label' />

    <Checkbox
        id='checkbox1'
        labelText='Checkbox label' />

    <Checkbox
        id='checkbox2'
        labelText='Checkbox label is really long and can wrap to multiple lines lorem ipsum dolar sit amet is really long and can wrap to multiple lines' />
    </div>
```

## Colors

By default, the checkboxes use the theme's primary color. But any other color may be passed in.

```.jsx
  <div>
    <Checkbox
        checked
        id='colorCheckbox'
        labelText='Default color' />
    <Checkbox
        checked
        color={magma.colors.success01}
        id='primaryColorCheckbox'
        labelText='Success color' />
    <Checkbox
        checked
        color={magma.colors.danger}
        id='dangerColorCheckbox'
        labelText='Danger color' />
    <Checkbox
        checked
        color={magma.colors.pop02}
        id='popColorCheckbox'
        labelText='Pop color' />
    </div>
```

## Disabled

Adding the optional boolean prop `disabled` will apply disabled styles and add the disabled attribute to the underlying HTML element.

```.jsx
  <div>
    <Checkbox
        disabled
        id='disabledCheckbox'
        labelText='Disabled checkbox' />
    </div>
```

## Indeterminate

In addition to being checked or unchecked, checkboxes also support an indeterminate or mixed state, but using the optional `indeterminate` boolean prop.

```.jsx
  <div>
    <Checkbox
        id='indeterminateCheckbox'
        indeterminate
        labelText='Indeterminate checkbox' />
    <Checkbox
        color={magma.colors.success01}
        id='indeterminateSuccessCheckbox'
        indeterminate
        labelText='Indeterminate checkbox' />
    </div>
```

## Inverse

Inverse is an optional boolean prop, that may be passed in when the component is to be displayed on a dark background.

```.jsx
  <div style={{ background : '#00263E', padding: '10px' }}>
    <Checkbox
        checked
        id='inverseCheckbox'
        inverse
      labelText='Inverse' />
    <Checkbox
        checked
        color={magma.colors.success01}
        id='successInverseCheckbox'
        inverse
        labelText='Success inverse' />
    <Checkbox
        checked
        color={magma.colors.danger}
        id='dangerInverseCheckbox'
        inverse
        labelText='Danger inverse' />
    <Checkbox
        checked
        color={magma.colors.pop02}
        id='popInverseCheckbox'
        inverse
        labelText='Pop inverse' />
      <Checkbox
          disabled
          id='disabledInverseCheckbox'
          inverse
          labelText='Disabled inverse' />
      <Checkbox
          id='indeterminateInverseCheckbox'
          indeterminate
          inverse
          labelText='Indeterminate checkbox' />
    </div>
```

## Visually Hidden Label

Adding the `textVisuallyHidden` prop will make the label text available to screen readers, but will hide it visually.

```.jsx
  <div>
    <Checkbox
        id='visuallyHiddenCheckbox'
        labelText='Disabled checkbox'
        textVisuallyHidden />
    </div>
```

## Custom Styles

Custom styles can be passed into the Checkbox component. The `containerStyle` property will apply to the container.
Additional `labelStyle` and `inputStyle` properties are available to style the respective elements. Please use discretion when adding custom styles.

```.jsx
  <div>
    <Checkbox
        checked
        containerStyle={{ 'border': '1px dotted purple'}}
        inputStyle={{ 'boxShadow': '3px 3px 3px rgba(0, 0 , 0, .6)'}}
        labelStyle={{ 'color': 'brown', 'fontStyle': 'italic'}}
        labelText='Custom styles' />

    <Checkbox
        checked
        labelText='No custom styles' />
  </div>
```

## Testing
