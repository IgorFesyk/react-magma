# Loading Indicators

## Basic Usage

```.jsx
    <LoadingIndicator />
```

## Custom Messages

```.jsx
    <LoadingIndicator
      message1="Custom message"
      message2="An additional custom message"
      message3="Third and final custom message" />
```
