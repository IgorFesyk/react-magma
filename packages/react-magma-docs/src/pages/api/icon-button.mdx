# Icon Buttons

## Basic Usage

Icon buttons have an icon included in them. If children are included with these buttons (such as text), then they render in a style similar to the non-icon buttons.
If no children are included, then they render in the icon-only style.

```.jsx
    <>
        <p>
          <IconButton aria-label="Icon" icon={<BellIcon />} />
          <IconButton aria-label="Icon" icon={<BellIcon />} />
          <IconButton aria-label="Icon" variant="outline" icon={<BellIcon />} />
          <IconButton aria-label="Icon" variant="link" icon={<BellIcon />} />
          <IconButton aria-label="Icon" color="secondary" icon={<Cog2Icon />} />
          <IconButton aria-label="Icon" color="secondary" variant="outline" icon={<Cog2Icon />} />
          <IconButton aria-label="Icon" color="secondary" variant="link" icon={<Cog2Icon />} />
          <IconButton aria-label="Icon" color="success" icon={<CheckIcon />} />
          <IconButton aria-label="Icon" color="success"variant="outline" icon={<CheckIcon />} />
          <IconButton aria-label="Icon" color="success" variant="link" icon={<CheckIcon />} />
          <IconButton aria-label="Icon" color="danger" icon={<AlertIcon />} />
          <IconButton aria-label="Icon" color="danger" variant="outline" icon={<AlertIcon />} />
          <IconButton aria-label="Icon" color="danger" variant="link" icon={<AlertIcon />} />
          <IconButton aria-label="Icon" disabled icon={<BellIcon />} />
        </p>
         <p>
          <IconButton icon={<BellIcon />}>Button with text</IconButton>
          <IconButton icon={<BellIcon />} color="secondary" variant="outline" >Button with text</IconButton>
          <IconButton icon={<BellIcon />} color="success" variant="link" >Button with text</IconButton>
        </p>
    </>
```

## Assign Icon to Variable

If you find JSX being passed in as a prop directly ugly, you can assign an icon to a variable and pass that in as the prop.

```.jsx
  (props) => {
    const icon = <BellIcon />

    return <IconButton aria-label="Icon" icon={icon} />
  }
```

## Custom Size

Icon buttons have default sizes for small, medium and large icon buttons respectively. However, if a pixel size is specified for an icon itself, that size will take take precedence.

```.jsx
  (props) => {
    const icon = <BellIcon size={10} />

    return <IconButton aria-label="Icon" icon={icon}>Custom Icon Size</IconButton>
  }
```

## Aria-Label

For icon only buttons you _MUST_ pass in an `aria-label` property to satisfy accessibility standards.

```.jsx
    <>
        <IconButton aria-label="Icon" icon={<BellIcon />} />
    </>
```

## Icon Position

Use the optional prop `iconPosition` to position the icon of the button on either side of the text.
The `iconPosition` property takes the values `left` and `right`.

```.jsx
    <>
        <IconButton icon={<BellIcon />}>Icon Left</IconButton>
        <IconButton color="secondary" icon={<InfoIcon />} iconPosition="right">Icon Right</IconButton>
    </>
```

## Sizes

Sizes for `IconButtons` include `small`, `medium`, and `large`, with `medium` being the default value.

```.jsx
    <>
      <p>
          <IconButton icon={<BellIcon />} size="small" />
          <IconButton icon={<BellIcon />} />
          <IconButton icon={<BellIcon />} size="large" />
      </p>
      <p>
          <IconButton icon={<BellIcon />} size="small" iconPosition="left">Small</IconButton>
          <IconButton icon={<BellIcon />} iconPosition="left">Medium</IconButton>
          <IconButton icon={<BellIcon />} size="large" iconPosition="left">Large</IconButton>
      </p>
    </>
```

## Shape

Icon only buttons have `shape = round`, by default.

```.jsx
    <>
      <IconButton icon={<BellIcon />} />
      <IconButton icon={<BellIcon />} shape="fill" />
      <IconButton icon={<BellIcon />} shape="rightCap" />
      <IconButton icon={<BellIcon />} shape="leftCap" />
    </>
```

## Block Buttons

Passing the `block` property will set the button to display full-width.
Block buttons only work when the `icon` property is set to `false` or not defined.

```.jsx
    <>
        <IconButton icon={<BellIcon />} block iconPosition="left">Block button</IconButton>
        <IconButton icon={<BellIcon />} color="secondary" block iconPosition="right">Block button</IconButton>
    </>
```

## Event Handling

The `onClick` event will fire when the button is clicked, or when the enter or space bar key is pressed.
Events do not fire on disabled buttons.

```.jsx
  (props) => {
    const [iconButtonCounter, setIconButtonCounter] = React.useState(0);
    const [textIconButtonCounter, setTextIconButtonCounter] = React.useState(0);

    return <>
      <p><strong>Icon Counter: </strong> {iconButtonCounter} | <strong>Text Icon Counter: </strong> {textIconButtonCounter}</p>
      <IconButton
        icon={<BellIcon />}
        onClick={() => {
          setIconButtonCounter(count => count + 1)
        }}>Click me</IconButton>
      <IconButton
        icon={<BellIcon />}
        textPosition="left"
        onClick={() => {
          setTextIconButtonCounter(count => count + 1)
        }}>Click me</IconButton>
      <IconButton
        disabled
        icon={<BellIcon />}
        onClick={() => {
          setIconButtonCounter(count => count + 1)
        }}>Click me if you can</IconButton>
    </>
  }
```
