## Basic Usage

The children of the tooltip, determines the content inside of the tooltip bubble.

The `trigger` prop must take a react element (not just a string), and determines the content that tiggers the tooltip on hover or on focus. Note: in order to trigger the content on focus (a requirement for accessibility), the `trigger` should be a focusable element.

```.jsx
    <>
        <Tooltip content="What's this?">
          <Button icon={<QuestionCircleIcon />} variant="link" />
        </Tooltip>

        <Tooltip content={<>Lorem ipsum <i>dolor sit amet</i>, mutat utroque. Vel molestie invidunt dissentiet no, ut vim iriure.</>}>
          <Button icon={<QuestionCircleIcon />} variant="link" />
        </Tooltip>
    </>
```

## Position

The `Tooltip` component takes a `position` property, that accepts the following values: bottom, left, right, top. Top is the default value.

```.jsx
    <>
        <Tooltip position='bottom' content='Tooltip bottom'>
          <Button aria-label="Bottom" icon={<CircleDownIcon />} variant="link" />
        </Tooltip>

        <Tooltip position='left' content='Tooltip left'>
          <Button aria-label="Left" icon={<CircleLeftIcon />} variant="link" />
        </Tooltip>

        <Tooltip position='right' content='Tooltip right'>
          <Button aria-label="Right" icon={<CircleRightIcon />} variant="link" />
        </Tooltip>

         <Tooltip position='top' content='Tooltip top'>
          <Button aria-label="Top" icon={<CircleUpIcon />} variant="link" />
        </Tooltip>
    </>
```

## Inverse

The `isInverse` property is an optional boolean, that reverses the colors so that the component can better appear on a dark background.
The default value is `false`.

```.jsx
    <div style={{ background: '#00263E', color: '#fff'}}>
        <Tooltip isInverse position='right' content='Inverse tooltip'>
          <Button aria-label="Question Icon" icon={<QuestionCircleIcon />} isInverse variant="link" />
        </Tooltip>
    </div>
```
