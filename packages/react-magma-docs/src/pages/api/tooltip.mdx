## Basic Usage

The children of the tooltip, determines the content inside of the tooltip bubble.

The `trigger` prop must take a react element (not just a string), and determines the content that tiggers the tooltip on hover or on focus. Note: in order to trigger the content on focus (a requirement for accessibility), the `trigger` should be a focusable element.

```.jsx
    <>
        <Tooltip trigger={<Button icon={<QuestionCircleIcon />} variant="link" />}>What's this?</Tooltip>
        <Tooltip trigger={<Button icon={<QuestionCircleIcon />} variant="link" />}>Lorem ipsum dolor sit amet, mutat utroque cu quo. Vel molestie invidunt dissentiet no, ut vim iriure eloquentiam.</Tooltip>
    </>
```

## Position

The `Tooltip` component takes a `position` property, that accepts the following values: bottom, left, right, top. Top is the default value.

```.jsx
    <>
        <Tooltip position='bottom' trigger={<Button aria-label="Question Icon" icon={<CircleDownIcon />} variant="link" />}>Tooltip bottom</Tooltip>
        <Tooltip position='left' trigger={<Button icon={<CircleLeftIcon />} variant="link" />}>Tooltip left</Tooltip>
        <Tooltip position='right' trigger={<Button icon={<CircleRightIcon />} variant="link" />}>Tooltip right</Tooltip>
        <Tooltip position='top' trigger={<Button icon={<CircleUpIcon />} variant="link" />}>Tooltip top</Tooltip>
        <br/>
        <Tooltip position='bottom' trigger={<Button aria-label="Question Icon" icon={<CircleDownIcon />} variant="link" />}>Lorem ipsum dolor sit amet, mutat utroque cu quo. Vel molestie invidunt dissentiet no, ut vim iriure eloquentiam.</Tooltip>
        <Tooltip position='left' trigger={<Button icon={<CircleLeftIcon />} variant="link" />}>Lorem ipsum dolor sit amet, mutat utroque cu quo. Vel molestie invidunt dissentiet no, ut vim iriure eloquentiam.</Tooltip>
        <Tooltip position='right' trigger={<Button icon={<CircleRightIcon />} variant="link" />}>Lorem ipsum dolor sit amet, mutat utroque cu quo. Vel molestie invidunt dissentiet no, ut vim iriure eloquentiam.</Tooltip>
        <Tooltip position='top' trigger={<Button icon={<CircleUpIcon />} variant="link" />}>Lorem ipsum dolor sit amet, mutat utroque cu quo. Vel molestie invidunt dissentiet no, ut vim iriure eloquentiam.</Tooltip>
    </>
```

## Inverse

The `Tooltip` component takes an optional `isInverse` prop, for when the tooltip is to appear on a dark background.

```.jsx
    <div style={{ background : '#00263E', color: '#ffffff', padding: '10px' }}>
        <Tooltip isInverse position="right" trigger={<Button aria-label="Question Icon" icon={<QuestionCircleIcon />} isInverse variant="link" />}>Inverse tooltip</Tooltip>
    </div>
```

## Refs

```.jsx
  (props) => {
    const tooltipRef = React.useRef();

    return (
      <div>
        <Tooltip ref={tooltipRef} position="top" trigger={<Button aria-label="Question Icon" icon={<QuestionCircleIcon />} variant="link" />}>Ref tooltip</Tooltip>
        <Button
        onClick={() => {
          tooltipRef.current.focus();
        }}>Click to focus the tooltip</Button>
      </div>
    )
  }
```
