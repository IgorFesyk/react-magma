# Spinner

Spinners indicate the loading state of a component or page. Spinners are used within the <Link to="/api/loading-indicator">Loading Indicator</Link> component.

## Basic Usage

```.jsx
<Spinner />
```

## Aria-Label

By default, the aria-label for the spinner is "Loading", but this can be customized using the `aria-label` prop.

```.jsx
<Spinner aria-label="Custom aria label" />
```

## Color

The default is `magma.colors.primary` (`#006298`), but can be customized using the `color` prop, which takes a string.

```.jsx
<Spinner color={magma.colors.danger} />
```

## Size

The default size is 15px, but can be customized using the `size` prop, which takes a number of pixels.

```.jsx
<Spinner size={30} />
```

## Spinner Props

**This component uses `forwardRef`. The ref is applied to the outer `span` element.**

All of the [global HTML attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes) can be provided as props and will be applied to the wrapping `span` element that gets rendered.
