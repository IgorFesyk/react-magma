# Icons

## Basic Usage

```.jsx
  <p>
    <span className='demo-icon-container'>
      <InfoIcon id="infoBasic" title="Info Available" />
    </span>
  </p>
```

## Sizes

```.jsx
  <p>
    <span className='demo-icon-container'>
      <AngleDownIcon id="smallSize" size={20} title="Small Size" />
    </span>
    <span className='demo-icon-container'>
      <AngleDownIcon id="largeSize" size={50} title="Large Size" />
    </span>
  </p>
```

## Color

```.jsx
  <p>
    <span className='demo-icon-container'>
      <AngleDownIcon id="noColor" title="No Color" />
    </span>
    <span className='demo-icon-container'>
      <AngleDownIcon id="someColor" color="green" title="Some Color" />
    </span>
  </p>
```

## Testing

Passing in the `testId` prop will add the `data-testid` attribute to the svg element for easier querying in tests.

```html
<svg
  data-testid="test-id"
  class="icon"
  height="24"
  width="24"
  fill="currentColor"
  xmlns="http://www.w3.org/2000/svg"
  viewBox="0 0 25.63 25.63"
  aria-labelledby="infoBasic"
>
  <title id="infoBasic">Info Available</title>
  <path
    d="M15.26,10.15V7.56H12.74v2.59Zm0,10.29v-7.7H12.74v7.7ZM14,1.18A12.82,12.82,0,1,1,1.18,14,12.79,12.79,0,0,1,14,1.18Z"
    transform="translate(-1.18 -1.18)"
  ></path>
</svg>
```
