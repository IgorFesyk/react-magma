# Icons

## Basic Usage

```.jsx
  <p>
    <span className='demo-icon-container'>
      <GroupIcon />
    </span>
  </p>
```

## Title

The `title` prop can be used to set the `<title>` element within the SVG.

The title/svg association will be handled via the passed in `id` if one is supplied or via an auto-generated `id` if not supplied. The `id` of the title is linked to the SVG with `aria-labelledby`,
so it will be read screen readers.

```.jsx
  <p>
    <span className='demo-icon-container'>
      <CrossIcon title="Close This Message" />
    </span>
    <span className='demo-icon-container'>
      <InfoIcon id="infoIcon" title="Info Available" />
    </span>
  </p>
```

## Sizes

```.jsx
  <p>
    <span className='demo-icon-container'>
      <AngleDownIcon size={20} title="Small Size" />
    </span>
    <span className='demo-icon-container'>
      <AngleDownIcon size={50} title="Large Size" />
    </span>
  </p>
```

## Color

```.jsx
  <p>
    <span className='demo-icon-container'>
      <AngleDownIcon title="No Color" />
    </span>
    <span className='demo-icon-container'>
      <AngleDownIcon color="green" title="Some Color" />
    </span>
  </p>
```

## Available Icons

```.jsx hideCode
    <div className='demo-icon-list-container'>{iconsArray.map((icon) => {
      const id = icon + 'Available';
      const title = icon + ' Title';
      const Icon = ICONS[icon];

      return <span className='demo-icon-container' key={id} >
        <Icon title={title} /><br />
        {icon}
      </span>
    })}</div>
```

## Testing

Passing in the `testId` prop will add the `data-testid` attribute to the svg element for easier querying in tests.

```html
<svg
  data-testid="test-id"
  class="icon"
  height="24"
  width="24"
  fill="currentColor"
  xmlns="http://www.w3.org/2000/svg"
  viewBox="0 0 25.63 25.63"
  aria-labelledby="infoBasic"
>
  <title id="infoBasic">Info Available</title>
  <path
    d="M15.26,10.15V7.56H12.74v2.59Zm0,10.29v-7.7H12.74v7.7ZM14,1.18A12.82,12.82,0,1,1,1.18,14,12.79,12.79,0,0,1,14,1.18Z"
    transform="translate(-1.18 -1.18)"
  ></path>
</svg>
```
