# Using Tabs

## Simple Example

This is a simple example of Tab Components

```.jsx
      function TabComponent() {
        const wrapperStyles = {
          backgroundColor: '#e6e6e6',
          padding: '25px'
        }

        const panelStyles = {
          height: '100px',
          padding: '5px',
          backgroundColor: '#fff'
        }

        return (
          <div id="example wrapper" style={wrapperStyles}>
            <TabsContainer>
                <Tabs
                  orientation='horizontal'
                >
                  <Tab>Main page</Tab>
                  <Tab>FAQ</Tab>
                  <Tab>About us</Tab>
                </Tabs>

                <TabPanel index={0}>
                  <div style={panelStyles}>Main page</div>
                </TabPanel>
                <TabPanel index={1}>
                  <div style={panelStyles}>FAQ</div>
                </TabPanel>
                <TabPanel index={2}>
                  <div style={panelStyles}>About us</div>
                </TabPanel>
            </TabsContainer>
          </div>
    )}

```

## Custom Logic

This is a simple example of Tab Components with custom switching logic

```.jsx
      class TabComponent extends React.Component {
        constructor(props) {
          super(props);
          this.state = {
            activeIndex: 0,
          }

          this.wrapperStyles = {
            backgroundColor: '#e6e6e6',
            padding: '25px'
          }

          this.panelStyles = {
            height: '100px',
            padding: '5px',
            backgroundColor: '#fff'
          }

          this.handleChange = this.handleChange.bind(this)
        }

        handleChange(index) {
          this.setState(() => ({
            activeIndex: index,
          }))
        }

        render() {
          return (
              <div id="example wrapper" style={this.wrapperStyles}>
                <TabsContainer>
                    <Tabs
                      orientation='horizontal'
                      activeIndex={this.state.activeIndex}
                      onChange={this.handleChange}
                    >
                      <Tab index={0}>First item</Tab>
                      <Tab index={1}>Second item</Tab>
                      <Tab index={2}>Third item</Tab>
                    </Tabs>

                    <TabPanel index={0} activeIndex={this.state.activeIndex}>
                      <div style={this.panelStyles}>First item</div>
                    </TabPanel>
                    <TabPanel index={1} activeIndex={this.state.activeIndex}>
                      <div style={this.panelStyles}>Second item</div>
                    </TabPanel>
                    <TabPanel index={2} activeIndex={this.state.activeIndex}>
                      <div style={this.panelStyles}>Third item</div>
                    </TabPanel>
                </TabsContainer>
              </div>
          )
        }
      }

```

## Nav Tabs

This is an example with custom element in Tabs.
By default we use "button" element in Tabs, but you can throw your custom element inside Tab component.
Please in this case put all children inside "component" prop instead of direct throwing in Tabs.
If you would like to use Tab like an route switcher keep "path" prop in Tab equal to desired path.

```.jsx
      function TabComponent() {
        const wrapperStyles = {
          backgroundColor: '#e6e6e6',
          padding: '25px'
        }

        const Link = ({children}) => {
          // This is just example of route component
          return <HyperLink style={{
            minWidth: '150px',
            fontSize: 'inherit',
            width: '100%',
            height: '100%',
            padding: '10px',
            color: 'inherit',
            opacity: 0.7,
            textDecoration: 'none'
          }}>{children}</HyperLink>
        }

        return (
          <div id="example wrapper" style={wrapperStyles}>
            <TabsContainer>
                <Tabs
                  orientation='horizontal'
                >
                  <Tab path="/main" component={<Link to="/main">Main page</Link>}/>
                  <Tab path="/faq" component={<Link to="/faq">FAQ</Link>}/>
                  <Tab path="/about" component={<Link to="/about">About us</Link>}/>
                </Tabs>
            </TabsContainer>
          </div>
    )}
```

## Inverse

The inverse property is an optional boolean, that reverses the colors so that the tabs can better appear on a dark background.
The default value is false.

### With blue Background

```.jsx
      function TabComponent() {
        const wrapperStyles = {
          backgroundColor: '#0f3765',
          padding: '25px'
        }

        return (
          <div id="example wrapper" style={wrapperStyles}>
            <TabsContainer>
                <Tabs
                  orientation='horizontal'
                  inverse={true}
                >
                 <Tab>First item</Tab>
                  <Tab>Second item</Tab>
                  <Tab>Third item</Tab>
                </Tabs>
            </TabsContainer>
          </div>
    )}
```
