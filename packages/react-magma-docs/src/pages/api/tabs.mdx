# Using Tabs

## Simple example

This is a simple example of Tab Components

```.jsx
      function TabComponent() {
        const wrapperStyles = {
          backgroundColor: '#e6e6e6',
          padding: '25px'
        }

        const panelStyles = {
          height: '100px',
          padding: '5px'
        }

        return (
          <div id="example wrapper" style={wrapperStyles}>
            <TabsContainer>
                <Tabs
                  orientation='horizontal'
                >
                  <Tab ariaLabel='first-tab'>Main page</Tab>
                  <Tab ariaLabel='second-tab'>FAQ</Tab>
                  <Tab ariaLabel='third-tab'>About us</Tab>
                </Tabs>

                <TabPanel index={0}>
                  <div style={panelStyles}>Main page</div>
                </TabPanel>
                <TabPanel index={1}>
                  <div style={panelStyles}>FAQ</div>
                </TabPanel>
                <TabPanel index={2}>
                  <div style={panelStyles}>About us</div>
                </TabPanel>
            </TabsContainer>
          </div>
    )}
```

## Custom logic

This is a simple example of Tab Components with custom switching logic

```.jsx
      class TabComponent extends React.Component {
        constructor(props) {
          super(props);
          this.state = {
            activeIndex: 0,
          }

          this.wrapperStyles = {
            backgroundColor: '#e6e6e6',
            padding: '25px'
          }

          this.panelStyles = {
            height: '100px',
            padding: '5px'
          }

          this.handleChange = this.handleChange.bind(this)
        }

        handleChange(index) {
          this.setState(() => ({
            activeIndex: index,
          }))
        }

        render() {
          return (
              <div id="example wrapper" style={this.wrapperStyles}>
                <TabsContainer>
                    <Tabs
                      activeIndex={this.state.activeIndex}
                      onChange={this.handleChange}
                    >
                      <Tab index={0}>First item</Tab>
                      <Tab index={1}>Second item</Tab>
                      <Tab index={2}>Third item</Tab>
                    </Tabs>

                    <TabPanel index={0} activeIndex={this.state.activeIndex}>
                      <div style={this.panelStyles}>First item</div>
                    </TabPanel>
                    <TabPanel index={1} activeIndex={this.state.activeIndex}>
                      <div style={this.panelStyles}>Second item</div>
                    </TabPanel>
                    <TabPanel index={2} activeIndex={this.state.activeIndex}>
                      <div style={this.panelStyles}>Third item</div>
                    </TabPanel>
                </TabsContainer>
              </div>
          )
        }
      }

```

## Nav tabs

This is an example with custom element in Tabs
By default we use "button" element in Tabs, but you can throw your custom element inside Tab component
Please in this case put all childrens inside "component" prop instead of direct throwing in Tabs

```.jsx
      function TabComponent() {
        const wrapperStyles = {
          backgroundColor: '#e6e6e6',
          padding: '25px'
        }

        const panelStyles = {
          height: '100px',
          padding: '5px'
        }

        return (
          <div id="example wrapper" style={wrapperStyles}>
            <TabsContainer>
                <Tabs
                  orientation='horizontal'
                >
                  <Tab ariaLabel='first-tab' component={<a href="/api/tabs">Main page</a>}></Tab>
                  <Tab ariaLabel='second-tab'>FAQ</Tab>
                  <Tab ariaLabel='third-tab'>About us</Tab>
                </Tabs>

                <TabPanel index={0}>
                  <div style={panelStyles}>Main page</div>
                </TabPanel>
                <TabPanel index={1}>
                  <div style={panelStyles}>FAQ</div>
                </TabPanel>

```
