# Theming

## Using Themes

By default, all components in the React Magma library use the Magma theme. However, you can pass in an alternative theme using the `<ThemeContext.Provider>`,
and passing in your custom theme as the `value` property. Any descendants of the context provider will receive the theme that is passed in.

Use the dropdown to change theme for the demo component.

```.jsx
  (props) => {
    const [theme, setTheme] = React.useState(magma);

    function handleItemChange(changes) {
      setTheme(changes.selectedItem.value);
    }

    return (
      <div>
        <Select
          id="themeSwitcher"
          labelText="Select Theme"
          onSelectedItemChange={handleItemChange}
          items={[{
            value: magma,
            label: 'Magma (default)'
          }, {
            value: amgam,
            label: 'Amgam'
          }, {
            value: amgamDark,
            label: 'Amgam Dark'
          }]} />

        <ThemeContext.Provider value={theme}>
          <DemoComponent idPrefix="theme1" />
        </ThemeContext.Provider>
      </div>
    )
  }
```
