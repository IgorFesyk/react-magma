# Hide at Breakpoint

The `HideAtBreakpoint` component can be used to quickly hide a component at a particular breakpoint.
It takes two props, `minWidth` and `maxWidth` for the minimum and maximum widths, specified in pixels.

## Basic Usage

```.jsx
    <>
     <HideAtBreakpoint maxWidth={600}>
            <p>This text will be visible when the browser width is 600px or wider.</p>
        </HideAtBreakpoint>
        <HideAtBreakpoint minWidth={600}>
             <p>This text will be visible when the browser width is 600px or smaller.</p>
        </HideAtBreakpoint>
        <HideAtBreakpoint>
             <p>This text will always be visible.</p>
        </HideAtBreakpoint>
    </>
```
