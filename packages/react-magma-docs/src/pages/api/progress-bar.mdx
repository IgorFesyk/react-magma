# Progress Bars

A progress bar can be used to show a user how far along they are in a process.

## Basic Usage

The `percentage` prop is used to display how much of the bar should be filled in. The default value is 0.

```.jsx
    <ProgressBar percentage={25} />
```

## Animated

```.jsx
    <ProgressBar animated percentage={25} />
```

## Background Color

The `bgColor` prop can be used to set the background color of the progress bar. The default value is `#006298`, or `magma.colors.primary`.

```.jsx
    <ProgressBar bgColor={magma.colors.danger} percentage={25} />
```

## Height

The `height` prop can be used to set the height of the progress bar in pixels. The default value is 16.

```.jsx
    <ProgressBar height={10} percentage={25} />
```

## Inverse

Inverse is an optional boolean prop, that may be passed in when the component is to be displayed on a dark background. When the `inverse` prop is used,
the default backgronud color of the progress bar is `#00A9E0` or `magma.colors.foundation03`.

```.jsx
    <div style={{ background : '#00263E', color: '#ffffff', padding: '10px' }}>
        <ProgressBar inverse percentage={25} />
    </div>
```

## Percentage Visible

If the `percentageVisible` prop is true, the percentage value will display to the right of the progress bar.

```.jsx
        <ProgressBar percentageVisible percentage={25} />
```
