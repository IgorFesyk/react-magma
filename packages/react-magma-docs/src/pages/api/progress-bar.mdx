A progress bar can be used to show a user how far along they are in a process. Progress bars are used within the <Link to="/api/loading-indicator">Loading Indicator</Link> component.

## Basic Usage

```.jsx
    <ProgressBar percentage={25} />
```

## Percentage

The `percentage` prop is used to display how much of the bar should be filled in. The default value is 0. This value can be adjusted dynamically.

```.jsx
  (props) => {
    const [percentage, setPercentage] = React.useState(0);

    return (
      <>
        <ProgressBar percentage={percentage} />
        <p/>
        <p>
          <Button disabled={percentage >= 100} onClick={() => setPercentage(percent => percent + 10)}>Increase</Button>
          <Button disabled={percentage <= 0} onClick={() => setPercentage(percent => percent - 10)}>Decrease</Button>
        </p>
      </>
    )
  }
```

## Animated

```.jsx
    <ProgressBar isAnimated percentage={25} />
```

## Background Color

The `color` prop can be used to set the background color of the progress bar. This prop takes the following values: `danger`, `primary`, `pop`, `pop02`, `success`.
The default value is `primary` which renders as `#006298`, or `magma.colors.primary`.

```.jsx
    <>
      <ProgressBar color='danger' percentage={25} /><br/>
      <ProgressBar color='pop' percentage={25} /><br/>
      <ProgressBar color='pop02' percentage={25} /><br/>
      <ProgressBar color='primary' percentage={25} /><br/>
      <ProgressBar color='success' percentage={25} /><br/>
    </>
```

## Height

The `height` prop can be used to set the height of the progress bar in pixels. The default value is 15.

```.jsx
    <ProgressBar height={10} percentage={25} />
```

## Inverse

`isInverse` is an optional boolean prop, that may be passed in when the component is to be displayed on a dark background. When the `isInverse` prop is used,
the default background color of the progress bar is `#00A9E0` or `magma.colors.foundation03`.

```.jsx
    <div style={{ background : '#00263E', color: '#ffffff', padding: '10px' }}>
        <ProgressBar isInverse percentage={25} />
    </div>
```

## Label Visible

If the `isLabelVisible` prop is true, the percentage value will display to the right of the progress bar.

```.jsx
        <ProgressBar isLabelVisible percentage={25} />
```

All of the [global HTML attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes) can be provided as props and will be applied to the wrapping `div` element that gets rendered.
