A progress bar can be used to show a user how far along they are in a process.

## Basic Usage

```.jsx
    <ProgressBar percentage={25} />
```

## Percentage

The `percentage` prop is used to display how much of the bar should be filled in. The default value is 0. This value can be adjusted dynamically.

```.jsx

    class Example extends React.Component {
      constructor(props) {
        super(props)
        this.state = {
          percentage: 0
        }
        this.increasePercentage = this.increasePercentage.bind(this);
        this.decreasePercentage = this.decreasePercentage.bind(this);

      }

      increasePercentage() {
        this.setState({percentage: this.state.percentage + 10})
      }

       decreasePercentage() {
        this.setState({percentage: this.state.percentage - 10})
      }

      render() {
        const { percentage } = this.state;


        return (
          <>
            <ProgressBar percentage={percentage} />
            <p/>
            <p>
              <Button disabled={percentage >= 100} onClick={this.increasePercentage}>Increase</Button>
              <Button disabled={percentage <= 0} onClick={this.decreasePercentage}>Decrease</Button>
            </p>
          </>
        )
      }
    }
```

## Animated

```.jsx
    <ProgressBar animated percentage={25} />
```

## Background Color

The `bgColor` prop can be used to set the background color of the progress bar. This prop takes the following values: `danger`, `primary`, `pop01`, `pop02`, `success`.
The default value is `primary` which renders as `#006298`, or `magma.colors.primary`.

```.jsx
    <>
      <ProgressBar bgColor='danger' percentage={25} /><br/>
      <ProgressBar bgColor='pop01' percentage={25} /><br/>
      <ProgressBar bgColor='pop02' percentage={25} /><br/>
      <ProgressBar bgColor='primary' percentage={25} /><br/>
      <ProgressBar bgColor='success' percentage={25} /><br/>
    </>
```

## Height

The `height` prop can be used to set the height of the progress bar in pixels. The default value is 15.

```.jsx
    <ProgressBar height={10} percentage={25} />
```

## Inverse

Inverse is an optional boolean prop, that may be passed in when the component is to be displayed on a dark background. When the `inverse` prop is used,
the default backgronud color of the progress bar is `#00A9E0` or `magma.colors.foundation03`.

```.jsx
    <div style={{ background : '#00263E', color: '#ffffff', padding: '10px' }}>
        <ProgressBar inverse percentage={25} />
    </div>
```

## Label Visible

If the `labelVisible` prop is true, the percentage value will display to the right of the progress bar.

```.jsx
        <ProgressBar labelVisible percentage={25} />
```
