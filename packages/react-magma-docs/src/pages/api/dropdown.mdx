# Dropdown

## Basic Usage

```.jsx
      <Dropdown>testing</Dropdown>
```

## Drop Direction

```.jsx
      <Dropdown dropDirection='up'>testing</Dropdown>
```

## Alignment

```.jsx
      <Dropdown alignment='right'>testing</Dropdown>
```

## Dropdown Props

**Any other props supplied will be provided to the wrapping `div` element.**
