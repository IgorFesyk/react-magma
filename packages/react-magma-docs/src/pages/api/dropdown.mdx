# Dropdown

## Basic Usage

The `Dropdown` component should container two children: the `DropdownToggle` component, and the `DropdownMenu` component.
The `DropdownToggle` is a button that handles the opening and closing of the menu. The `DropdownMenu` component contains individual menu items,
represented in `DropdownMenuItem` components.

```.jsx
      <Dropdown>
         <DropdownToggle>Basic Dropdown</DropdownToggle>
          <DropdownMenu>
            <DropdownMenuItem>Menu item 1</DropdownMenuItem>
            <DropdownMenuItem>Menu item number two</DropdownMenuItem>
          </DropdownMenu>
      </Dropdown>
```

## Split Toggle

Instead of using the `DropdownToggle`, a `DropdownSplitToggle` can be used instead. When using this, the larger part of the button will accept an `onClick` event, and only the caret icon will open and close the menu.

```.jsx
      <Dropdown>
         <DropdownSplitToggle onClick={()=>{alert('Button Clicked!')}}>Split Toggle</DropdownSplitToggle>
          <DropdownMenu>
            <DropdownMenuItem>Menu item 1</DropdownMenuItem>
            <DropdownMenuItem>Menu item number two</DropdownMenuItem>
          </DropdownMenu>
      </Dropdown>
```

## Drop Direction

Using the `dropDirection` prop will change where the menu appears. Options are `down` and `up`, with `down` being default.

```.jsx
      <Dropdown dropDirection='up'>
          <DropdownToggle>Drop up</DropdownToggle>
          <DropdownMenu>
            <DropdownMenuItem>Menu item 1</DropdownMenuItem>
            <DropdownMenuItem>Menu item number two</DropdownMenuItem>
          </DropdownMenu>
      </Dropdown>
```

## Alignment

Using the `alignment` prop will the alignment of the menu. Options are `left` and `right`, with `left` being default.

```.jsx
      <Dropdown alignment='right'>
        <DropdownToggle>Right Aligned</DropdownToggle>
        <DropdownMenu>
          <DropdownMenuItem>Menu item 1</DropdownMenuItem>
          <DropdownMenuItem>Menu item number two</DropdownMenuItem>
        </DropdownMenu>
      </Dropdown>
```

## Fixed Width

By default, a dropdown menu will be as wide as the largest item. However, you may limit the width using the `width` prop, in which case items will wrap if needed.

```.jsx
      <Dropdown width="250px">
        <DropdownToggle>Fixed Width</DropdownToggle>
        <DropdownMenu>
          <DropdownMenuItem>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla tellus turpis, finibus et bibendum eget, rhoncus ut risus. Proin nec nisi ut eros aliquam interdum.</DropdownMenuItem>
          <DropdownMenuItem>Nunc egestas ullamcorper finibus. Phasellus non risus id mauris dignissim molestie in sit amet enim. In condimentum fringilla sodales.</DropdownMenuItem>
        </DropdownMenu>
      </Dropdown>
```

## Active Item Index

To set a dropdown item as active, provide each `DropdownMenuItem` with an `index` prop, and provide the `Dropdown` with an `activeIndex` prop.
When another item is clicked, that item will become active and the menu will remain open, rather than closing as usual.

```.jsx
    (props) => {

      const [activeValue, setActiveValue] = React.useState('two');

      function changeActiveValue(value) {
        setActiveValue(value);
      }

      return (
        <>
        <p><strong>Active Value: </strong> <span>{activeValue}</span></p>

        <Dropdown activeIndex={1}>
          <DropdownToggle>Selected Item</DropdownToggle>
          <DropdownMenu>
            <DropdownMenuItem index={0} onClick={changeActiveValue} value='one'>Dropdown Item 1</DropdownMenuItem>
            <DropdownMenuItem index={1} onClick={changeActiveValue} value='two'>Dropdown Item 2</DropdownMenuItem>
            <DropdownMenuItem index={2} onClick={changeActiveValue} value='three'>Dropdown Item 3</DropdownMenuItem>
          </DropdownMenu>
        </Dropdown>
        </>
    )
}
```

## Max-Height

By default, the max-height of a dropdown menu is 250px. That can be change by using the `maxHeight` prop.

```.jsx

        <DropdownToggle>Max-Height</DropdownToggle>
        <DropdownMenu>
            <DropdownMenuItem>Menu item 1</DropdownMenuItem>
            <DropdownMenuItem>Menu item 2</DropdownMenuItem>
            <DropdownMenuItem>Menu item 3</DropdownMenuItem>
```
