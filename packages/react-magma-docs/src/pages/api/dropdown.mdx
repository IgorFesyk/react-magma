# Dropdown

## Basic Usage

The `Dropdown` component should contain two children: the `DropdownToggle` component, and the `DropdownMenu` component.
The `DropdownToggle` is a button that handles the opening and closing of the menu. The `DropdownMenu` component contains individual menu items,
represented in `DropdownMenuItem` components.

```.jsx
      <Dropdown>
         <DropdownToggle>Basic Dropdown</DropdownToggle>
          <DropdownMenu>
            <DropdownMenuItem>Menu item 1</DropdownMenuItem>
            <DropdownMenuItem>Menu item number two</DropdownMenuItem>
          </DropdownMenu>
      </Dropdown>
```

## Split Toggle

Instead of using the `DropdownToggle`, a `DropdownSplitToggle` can be used instead. When using this, the larger part of the button will accept an `onClick` event, and only the caret icon will open and close the menu.

```.jsx
      <Dropdown>
         <DropdownSplitToggle onClick={()=>{alert('Button Clicked!')}}>Split Toggle</DropdownSplitToggle>
          <DropdownMenu>
            <DropdownMenuItem>Menu item 1</DropdownMenuItem>
            <DropdownMenuItem>Menu item number two</DropdownMenuItem>
          </DropdownMenu>
      </Dropdown>
```

## Drop Direction

Using the `dropDirection` prop will change where the menu appears. Options are `down` and `up`, with `down` being default.

```.jsx
      <Dropdown dropDirection='up'>
          <DropdownToggle>Drop up</DropdownToggle>
          <DropdownMenu>
            <DropdownMenuItem>Menu item 1</DropdownMenuItem>
            <DropdownMenuItem>Menu item number two</DropdownMenuItem>
          </DropdownMenu>
      </Dropdown>
```

## Alignment

Using the `alignment` prop will set the alignment of the menu. Options are `left` and `right`, with `left` being default.

```.jsx
      <Dropdown alignment='right'>
        <DropdownToggle>Right Aligned</DropdownToggle>
        <DropdownMenu>
          <DropdownMenuItem>Menu item 1</DropdownMenuItem>
          <DropdownMenuItem>Menu item number two</DropdownMenuItem>
        </DropdownMenu>
      </Dropdown>
```

## Fixed Width

By default, a dropdown menu will be as wide as the largest item. However, you may limit the width using the `width` prop, in which case items will wrap if needed.

```.jsx
      <Dropdown width="250px">
        <DropdownToggle>Fixed Width</DropdownToggle>
        <DropdownMenu>
          <DropdownMenuItem>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Nulla tellus turpis, finibus et bibendum eget, rhoncus ut risus. Proin nec nisi ut eros aliquam interdum.</DropdownMenuItem>
          <DropdownMenuItem>Nunc egestas ullamcorper finibus. Phasellus non risus id mauris dignissim molestie in sit amet enim. In condimentum fringilla sodales.</DropdownMenuItem>
        </DropdownMenu>
      </Dropdown>
```

## Disabled Item

```.jsx
      <Dropdown>
        <DropdownToggle>Disabled Item</DropdownToggle>
        <DropdownMenu>
          <DropdownMenuItem  onClick={()=>{alert('Item 1 clicked')}}>Menu item 1</DropdownMenuItem>
          <DropdownMenuItem  isDisabled onClick={()=>{alert('Item 2 clicked')}}>Menu item 2</DropdownMenuItem>
          <DropdownMenuItem  onClick={()=>{alert('Item 3 clicked')}}>Menu item 3</DropdownMenuItem>
        </DropdownMenu>
      </Dropdown>
```

## Custom Toggle Button

Both the `DropdownToggle` and ``DropdownSplitToggle`can be customized by using any of the props that the <Link to="/api/button">Button</Link> component accepts. If no`icon`property is provided to the`DropdownToggle`, the caret icon will appear.

```.jsx
      <>
        <Dropdown>
          <DropdownToggle size="small" color='marketing'>Extra Props Example 1</DropdownToggle>
            <DropdownMenu>
              <DropdownMenuItem>Menu item 1</DropdownMenuItem>
              <DropdownMenuItem>Menu item number two</DropdownMenuItem>
            </DropdownMenu>
        </Dropdown>
        <Dropdown>
          <DropdownSplitToggle size='large' variant='outline' color='danger'>Extra Props Split</DropdownSplitToggle>
            <DropdownMenu>
              <DropdownMenuItem>Menu item 1</DropdownMenuItem>
              <DropdownMenuItem>Menu item number two</DropdownMenuItem>
            </DropdownMenu>
        </Dropdown>
      </>
```

## Custom Toggle Icon Button

Similarly, the `DropdownToggle` can be customized by using any of the props that the <Link to="/api/icon-button">Icon Button</Link> component accepts, to create a button with an icon.

```.jsx
      <>
        <Dropdown>
          <DropdownToggle aria-label="Extra icon example" color='secondary' size='large' variant='link' icon={<Cog2Icon />} />
            <DropdownMenu>
              <DropdownMenuItem>Menu item 1</DropdownMenuItem>
              <DropdownMenuItem>Menu item number two</DropdownMenuItem>
            </DropdownMenu>
        </Dropdown>
        <Dropdown>
          <DropdownToggle aria-label="Extra icon example"  color='success' size='large' icon={<MenuIcon />} />
            <DropdownMenu>
              <DropdownMenuItem>Menu item 1</DropdownMenuItem>
              <DropdownMenuItem>Menu item number two</DropdownMenuItem>
            </DropdownMenu>
        </Dropdown>
         <Dropdown>
          <DropdownToggle variant="outline" size='small' icon={<Cog2Icon />}>Icon with Text</DropdownToggle>
            <DropdownMenu>
              <DropdownMenuItem>Menu item 1</DropdownMenuItem>
              <DropdownMenuItem>Menu item number two</DropdownMenuItem>
            </DropdownMenu>
        </Dropdown>
      </>
```

```.jsx

        <DropdownMenu>
          <DropdownMenuItem icon={<AsteriskIcon />}>Menu item 1</DropdownMenuItem>
          <DropdownMenuItem icon={<BellIcon />}>Menu item 2</DropdownMenuItem>

```

## Breaking a Menu Into Sections

A `DropdownMenu` can be broken into multiple sections, by using the `DropdownMenuHeader` and `DropdownMenuDivider` components.

```.jsx
      <Dropdown>
        <DropdownToggle>Divider and headings</DropdownToggle>
        <DropdownMenu>
          <DropdownMenuHeader>Section title A</DropdownMenuHeader>
          <DropdownMenuItem>Menu item 1</DropdownMenuItem>
          <DropdownMenuItem>Menu item 2</DropdownMenuItem>
          <DropdownMenuDivider />
          <DropdownMenuHeader>Section title B</DropdownMenuHeader>
          <DropdownMenuItem>Menu item 3</DropdownMenuItem>
          <DropdownMenuItem>Menu item 4</DropdownMenuItem>
          <DropdownMenuDivider />
          <DropdownMenuItem>Log out</DropdownMenuItem>
        </DropdownMenu>
      </Dropdown>
```

## Active Item Index

The `activeIndex` prop can be used to set an item within the dropdown as active.
When another item is selected, that item will become active and the menu will remain open, rather than closing as usual.

```.jsx
    (props) => {

      const [activeValue, setActiveValue] = React.useState('two');

      function changeActiveValue(value) {
        setActiveValue(value);
      }

      return (
        <>
        <p><strong>Active Value: </strong> <span>{activeValue}</span></p>
         <Dropdown activeIndex={1}>
          <DropdownToggle>Selected Item</DropdownToggle>
          <DropdownMenu>
            <DropdownMenuItem onClick={changeActiveValue} value='one'>Dropdown Item 1</DropdownMenuItem>
            <DropdownMenuItem onClick={changeActiveValue} value='two'>Dropdown Item 2</DropdownMenuItem>
            <DropdownMenuItem onClick={changeActiveValue} value='three'>Dropdown Item 3</DropdownMenuItem>
          </DropdownMenu>
        </Dropdown>
        </>
    )
}
```

## Max-Height

By default, the max-height of a dropdown menu is 250px. That can be changed by using the `maxHeight` prop.

```.jsx
      <Dropdown maxHeight='150px'>
        <DropdownToggle>Max-Height</DropdownToggle>
        <DropdownMenu>
            <DropdownMenuItem>Menu item 1</DropdownMenuItem>
            <DropdownMenuItem>Menu item 2</DropdownMenuItem>
            <DropdownMenuItem>Menu item 3</DropdownMenuItem>
            <DropdownMenuItem>Menu item 4</DropdownMenuItem>
            <DropdownMenuItem>Menu item 5</DropdownMenuItem>
            <DropdownMenuItem>Menu item 6</DropdownMenuItem>
            <DropdownMenuItem>Menu item 7</DropdownMenuItem>
            <DropdownMenuItem>Menu item 8</DropdownMenuItem>
            <DropdownMenuItem>Menu item 9</DropdownMenuItem>
            <DropdownMenuItem>Menu item 10</DropdownMenuItem>
        </DropdownMenu>
      </Dropdown>
```

## Dropdown with Form

Dropdowns can contain elements other than Dropdown Menu Items. The standard keyboard behavior would apply in these cases.

```.jsx
      <Dropdown width='300px' maxHeight='600px'>
        <DropdownToggle>Dropdown with Form</DropdownToggle>
        <DropdownMenu>
          <form style={{padding: '20px'}}>
              <Input type="email" labelText="Email Address" />
              <PasswordInput labelText="Password" />
              <div style={{textAlign: 'center'}}>
                <p>By signing in, you agree to our <a href="#">Terms of use</a>.</p>
                <Button isFullWidth>Sign In</Button>
                <p><a>Forgot password?</a></p>
              </div>
          </form>
          <DropdownMenuDivider />
          <div style={{textAlign: 'center', padding: '20px'}}>
            <p>
              Don't have an account? <br/>
              <a href="#">Register today!</a>
            </p>
          </div>
        </DropdownMenu>
      </Dropdown>
```

## Dropdown with Form 2

Another example of form elements in a dropdown.

```.jsx
      () => {
        const labelStyle = { justifyContent: 'space-between', width: '100%' };

        return (
            <Dropdown maxHeight='600px'>
              <DropdownToggle>Dropdown with Form 2</DropdownToggle>
              <DropdownMenu>
                <form style={{margin: 0}}>
                  <DropdownMenuHeader id='checkboxGroup'>Checkbox Group Label</DropdownMenuHeader>
                  <FormGroup labelledById='checkboxGroup'>
                      <Checkbox labelText={<><AsteriskIcon size={20} />Option 1</>} labelStyle={labelStyle} textPosition="left" />
                      <Checkbox labelText={<><BellIcon size={20} />Option 2</>} labelStyle={labelStyle} textPosition="left"  />
                  </FormGroup>
                  <DropdownMenuDivider />
                  <DropdownMenuHeader id='radioGroup'>Radio Group Label</DropdownMenuHeader>
                  <RadioGroup labelledById='radioGroup' name='radioGroupExample' value='1'>
                      <Radio labelText='Option 1' value='1' labelStyle={labelStyle} textPosition="left" />
                      <Radio labelText='Option 2' value='2' labelStyle={labelStyle} textPosition="left" />
                      <Radio labelText='Option 3' value='3' labelStyle={labelStyle} textPosition="left"  />
                  </RadioGroup>
                </form>
              </DropdownMenu>
            </Dropdown>
        )
      }
```
