# Toasts

`Toast` components use the <Link to="/api/alert">Alert component</Link> internally for it's content and styles. Any props that can be passed to an `Alert` can be passed to a `Toast` component.

## Basic Usage

Variants for toasts include `info`, `success`, `warning`, and `danger`, with `info` being the default value.

```.jsx
  (props) => {
    const [showDefaultAlert, setShowDefaultAlert] = React.useState(false);
    const [showInfoAlert, setShowInfoAlert] = React.useState(false);
    const [showSuccessAlert, setShowSuccessAlert] = React.useState(false);
    const [showWarningAlert, setShowWarningAlert] = React.useState(false);
    const [showDangerAlert, setShowDangerAlert] = React.useState(false);

    return (<>
      <Button size="small" onClick={() => setShowDefaultAlert(true)}>Show Default Toast</Button>
      <Button size="small" onClick={() => setShowInfoAlert(true)}>Show Info Toast</Button>
      <Button size="small" onClick={() => setShowSuccessAlert(true)}>Show Success Toast</Button>
      <Button size="small" onClick={() => setShowWarningAlert(true)}>Show Warning Toast</Button>
      <Button size="small" onClick={() => setShowDangerAlert(true)}>Show Danger Toast</Button>
      {showDefaultAlert && <Toast onDismiss={() => setShowDefaultAlert(false)}>Default Toast</Toast>}
      {showInfoAlert && <Toast variant="info" onDismiss={() => setShowInfoAlert(false)}>Info Toast</Toast>}
      {showSuccessAlert && <Toast variant="success" onDismiss={() => setShowSuccessAlert(false)}>Success Toast</Toast>}
      {showWarningAlert && <Toast variant="warning" onDismiss={() => setShowWarningAlert(false)}>Warning Toast</Toast>}
      {showDangerAlert && <Toast variant="danger" onDismiss={() => setShowDangerAlert(false)}>Danger Toast</Toast>}
    </>)
  }
```

## Accessibility Considerations

Toasts are intended to be small interruptions to users, so when a `Toast` component is added to a screen, it is important that screen readers or similar assistive technology
read the new content. One way to do this is to the use the <Link to="/api/announce">Announce component</Link>, which employs the `aria-live` attribute. Be sure to place
the conditional logic to display the Toast _inside_ the Announce component.

While toasts appear visually in the lower right side of the screen, they will exist in the dom where ever they are placed. Keep this in mind when you are structuring your
markup, to ensure that they have the expected tab order.

```.jsx
  (props) => {
    const [showAlert, setShowAlert] = React.useState(false);

    return (<>
      <Button onClick={() => setShowAlert(true)}>Show Toast in Announce</Button>
      <Announce>
        {showAlert && <Toast dismissible onDismiss={() => setShowAlert(false)}>Dismissible Toast with <a href="#">linked text</a></Toast>}
      </Announce>
    </>)
  }
```

## Toast Duration

```.jsx
  (props) => {
    const [showAlert, setShowAlert] = React.useState(false);

    return (<>
      <Button onClick={() => setShowAlert(true)}>Show Toast</Button>
      {showAlert && <Toast toastDuration={1000} onDismiss={() => setShowAlert(false)}>Quick Toast</Toast>}
    </>)
  }
```

## Dismissible

```.jsx
  (props) => {
    const [showAlert, setShowAlert] = React.useState(false);

    return (<>
      <Button onClick={() => setShowAlert(true)}>Show Dismissible Toast</Button>
      {showAlert && <Toast isDismissible onDismiss={() => setShowAlert(false)}>Dismissible Toast</Toast>}
    </>)
  }
```

## Disable Auto Dismiss

```.jsx
  (props) => {
    const [showAlert, setShowAlert] = React.useState(false);

    return (<>
      <Button onClick={() => setShowAlert(true)}>Show Toast</Button>
      {showAlert && <Toast isDismissible disableAutoDismiss={true} onDismiss={() => setShowAlert(false)}>Non-Auto Hiding Dismissible Toast</Toast>}
    </>)
  }
```

## Mouse Events

When you hover over the Toast the timer is paused. Once the mouse leaves the Toast the timer is restarted and runs for half of the original toast duration.

```.jsx
  (props) => {
    const [showAlert, setShowAlert] = React.useState(false);
    const [alertContent, setAlertContent] = React.useState('Default Alert Content');

    return (<>
      <Button onClick={() => setShowAlert(true)}>Show Toast</Button>
      {showAlert && <Toast
                      onDismiss={() => setShowAlert(false)}
                      onMouseEnter={() => setAlertContent('Mouse is hovering toast')}
                      onMouseLeave={() => setAlertContent('Mouse has stopped hovering toast')}>{alertContent}</Toast>}
    </>)
  }
```

## Custom Styles

Custom styles can be added to a toast using two different props. The `containerStyle` prop controls the style for the toast container itself,
which includes the component's positioning and z-index. The `alertStyle` prop controls the style for the interal alert component, which
includes the element's background, border-radius, and fade animation.

```.jsx
  (props) => {
    const [showAlert, setShowAlert] = React.useState(false);

    return (<>
      <Button onClick={() => setShowAlert(true)}>Show Toast</Button>
      {showAlert &&
        <Toast
          alertStyle={{border: '4px dotted yellow'}}
          containerStyle={{'border': '1px solid purple', 'left': 'auto', 'right': '20px'}}
          onDismiss={() => setShowAlert(false)}>Toast with custom styles</Toast>
      }
    </>)
  }
```
