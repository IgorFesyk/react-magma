# Span

The `Span` component is used to present inline text in a clear and consistent manner, by limiting the number to sizes and styles with a UI.

## Basic Usage

```.jsx
    <Span>
        Basic span component.
    </Span>
```

## Color

The `color` prop can be used to change the style or emphasis on the different variants.
It accepts the values: `success`, `danger` and `subdued`.

There are no inverse styles for the `success` or `danger` colors.

```.jsx
    <>
        <Paragraph>
            Paragraph with some <Span color="success">success text</Span>. Lorem ipsum dolar sit amet.
        </Paragraph>

        <Paragraph>
            Paragraph with some <Span color="subdued">subdued text</Span>. Lorem ipsum dolar sit amet.
        </Paragraph>

       <Paragraph>
            Paragraph with some <Span color="danger">danger text</Span>. Lorem ipsum dolar sit amet.
        </Paragraph>
    </>
```

## Inverse

The `isInverse` property is an optional boolean, that changes the colors so component can appear on a dark background.

There are no inverse styles for the `success` or `danger` colors.

```.jsx
    <div style={{background: '#003865', padding: '4px 16px'}}>
         <Paragraph isInverse>
            Inverse paragraph with <Span isInverse>some inverse text</Span>.  Lorem ipsum dolar sit amet.
        </Paragraph>
        <Paragraph isInverse>
            Inverse paragraph with <Span color="subdued" isInverse>some inverse, subdued text</Span>.  Lorem ipsum dolar sit amet.
        </Paragraph>
    </div>
```
