## Basic Usage

```.jsx
  <>
    <DownshiftSelect
      id="basicSelectId"
      name="basic"
      labelText="Basic"
      items={['Red', 'Blue', 'Green']}
    />
  </>
```

## Combobox

```.jsx
  <>
    <DownshiftSelect
      id="comboboxId"
      name="combobox"
      labelText="Combobox"
      items={['Red', 'Blue', 'Green']}
      type='combo'
    />
  </>
```
