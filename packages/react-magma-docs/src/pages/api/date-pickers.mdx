## Basic Usage

```.jsx
    <DatePicker labelText="Date" />
```

## Default Date

By default, no date value is selected. You can add a default date, by using the `defaultDate` prop.

```.jsx
    <DatePicker defaultDate='07/23/2019' labelText="Date" />
```

## Placeholder Text

By default, the placeholder text of the input is 'MM/DD/YYYY'. You can override this by using the `placeholder` prop.
Note: the expected date format should appear somewhere in either the labelText or the placeholder.

```.jsx
    <DatePicker placeholder='Your birthday (format MM/DD/YYYY)' labelText="Birthday" />
```

## Helper Message

```.jsx
    <DatePicker labelText="Help Me" helperMessage="This is your help" />
```

## Error Message

```.jsx
    <DatePicker labelText="Error *" errorMessage="You made an error" required />
```

## Date Change Event

```.jsx
  (props) => {
    const [chosenDate, setChosenDate] = React.useState(0);
    const [inputValue, setInputValue] = React.useState('');

    return (<div>
      <p>
        <strong>Chosen Date: </strong>
        {chosenDate &&
          <span>
            {`${chosenDate.getMonth() + 1}/${chosenDate.getDate()}/${chosenDate.getFullYear()}`}
          </span>
        }
      </p>
      <p><strong>Input Value: </strong>{inputValue}</p>
      <DatePicker
        onInputChange={(event) => {
          setInputValue(event.target.value);
        }}
        onDateChange={(newChosenDate) => {
          setChosenDate(newChosenDate);
        }}
        onInputBlur={(chosenDate) => {
          console.log('blurred')
        }} />
    </div>);
  }
```

## Keyboard Navigation

The `DatePicker` is keyboard navigable for allowing the user to navigte between days, weeks, and months. If you would like to see all of the options for keyboard navigation click on the `?` button in the bottom right corner of the `DatePicker` or press the `?` key.

```.jsx
    <DatePicker labelText="Date" />
```
