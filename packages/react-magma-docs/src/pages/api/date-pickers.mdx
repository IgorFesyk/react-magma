## Basic Usage

```.jsx
    <DatePicker labelText="Date" />
```

## Click Events

```.jsx
  class Example extends React.Component {
    constructor(props) {
      super(props)
      this.state = {
        chosenDate: 0
      }
    }

    render() {
      const { chosenDate } = this.state;
      return <div>
      <p>
        <strong>Chosen Date: </strong>
        {chosenDate &&
          <span>
            {`${chosenDate.getMonth() + 1}/${chosenDate.getDate()}/${chosenDate.getFullYear()}`}
          </span>
        }
      </p>
      <DatePicker
        onDayClick={(chosenDate) => {
          this.setState(state => ({chosenDate}))
        }} />
      </div>
    }
  }
```

## Keyboard Navigation

The `DatePicker` is keyboard navigable for allowing the user to navigte between days, weeks, and months. If you would like to see all of the options for keyboard navigation click on the `?` button in the bottom right corner of the `DatePicker` or press the `?` key.

```.jsx
    <DatePicker labelText="Date" />
```
