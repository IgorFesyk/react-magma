## Basic usage

```.jsx
    <DatePicker labelText="Date" />
```

## Default date

By default, no date value is selected. You can add a default date, by using the `defaultDate` prop.

```.jsx
    <DatePicker defaultDate='07/23/2019' labelText="Date" />
```

## Placeholder text

By default, the placeholder text of the input is 'MM/DD/YYYY'. You can override this by using the `placeholderText` prop.
Note: the expected date format should appear somewhere in either the labelText or the placeholderText.

```.jsx
    <DatePicker placeholderText='Your birthday (format MM/DD/YYYY)' labelText="Birthday" />
```

## Helper message

```.jsx
    <DatePicker labelText="Help Me" helperMessage="This is your help" />
```

## Error message

```.jsx
    <DatePicker labelText="Error" errorMessage="You made an error" />
```

## Date change event

```.jsx
  class Example extends React.Component {
    constructor(props) {
      super(props)
      this.state = {
        chosenDate: 0,
        inputValue: ''
      }
    }

    render() {
      const { chosenDate, inputValue } = this.state;
      return <div>
      <p>
        <strong>Chosen Date: </strong>
        {chosenDate &&
          <span>
            {`${chosenDate.getMonth() + 1}/${chosenDate.getDate()}/${chosenDate.getFullYear()}`}
          </span>
        }
      </p>
      <p><strong>Input Value: </strong>{inputValue}</p>
      <DatePicker
        onInputChange={(event) => {
          this.setState({inputValue: event.target.value});
        }}
        onDateChange={(chosenDate) => {
          this.setState(state => ({chosenDate}))
        }}
        onInputBlur={(chosenDate) => {
          console.log('blurred')
        }} />
      </div>
    }
  }
```

## Keyboard navigation

The `DatePicker` is keyboard navigable for allowing the user to navigte between days, weeks, and months. If you would like to see all of the options for keyboard navigation click on the `?` button in the bottom right corner of the `DatePicker` or press the `?` key.

```.jsx
    <DatePicker labelText="Date" />
```
