## Usage
Headings used in a traditional heading/subheading/paragraph context should use the default styles defined in Magma and not be customized without good reason.

## Principles

### Consistent
The default styling of headings should be used consistently across pages when applied to a traditional set of content.

### Hierarchical
Headings provide a hierarchy and structure to a page, which is important for both users and screen readers.
