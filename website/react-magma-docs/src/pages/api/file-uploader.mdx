Upload files

## Basic Usage

```typescript
import React from 'react';
import { Fileuploader } from '@react-magma/file-uploader';

export function Example() {
  return <FileUploader 
    thumbnails={false}
    accept={['.png', '.jpg', '.svg']}
    maxFiles={5}
    maxSize={1024*1024}
    sendFiles={true}
    labelText="Upload files"
    helperMessage="Only PNG, JPG, and SVG files with a max size of 1MB"
  />
}
```
