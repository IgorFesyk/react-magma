<DocsHeading type="api">AppBar</DocsHeading>

The `AppBar` component is a relatively simple container, rendering a `header` element, used to display information and actions relating to the current screen.

It is the foundation for the <Link to="/patterns/header">Header Pattern</Link>, a more robust, more opinionated component that also handles the rendering of several components inside of it.

## Basic Usage

```typescript
import React from 'react';
import { AppBar } from 'react-magma-dom';

export function Example() {
  return <AppBar>AppBar content</AppBar>;
}
```

## Compact

The `isCompact` prop will render a shorter appbar with less padding.

```typescript
import React from 'react';
import { AppBar } from 'react-magma-dom';

export function Example() {
  return <AppBar isCompact>AppBar content</AppBar>;
}
```

## Inverse

The `isInverse` prop will render an appbar with a dark background and light text. Components inside will also need to have the `isInverse` prop added to them.

```typescript
import React from 'react';
import { AppBar } from 'react-magma-dom';

export function Example() {
  return <AppBar isInverse>AppBar content</AppBar>;
}
```

## Nav Tabs

This is a more complex example of putting mutliple elements inside an `AppBar`.

```typescript
import React from 'react';
import {
  AppBar,
  NavTabs,
  NavTab,
  Search,
  TabsIconPosition,
  magma,
} from 'react-magma-dom';
import { ImageIcon, FavoriteIcon, WorkIcon } from 'react-magma-icons';

export function Example() {
  return (
    <AppBar isInverse>
      <strong
        style={{
          alignItems: 'center',
          display: 'flex',
          paddingRight: magma.spaceScale.spacing12,
          whiteSpace: 'nowrap',
        }}
      >
        <ImageIcon
          size={24}
          style={{ marginRight: magma.spaceScale.spacing03 }}
        />
        LOGO
      </strong>
      <Search
        containerStyle={{ marginRight: magma.spaceScale.spacing06 }}
        isInverse
        onSearch={() => {}}
        placeholder="Search for content..."
      />
      <NavTabs
        aria-label="Navigation"
        backgroundColor="transparent"
        isInverse
        iconPosition={TabsIconPosition.left}
      >
        <NavTab icon={<FavoriteIcon />} isActive to="#">
          Favorites
        </NavTab>
        <NavTab icon={<WorkIcon />} to="#">
          Workspace
        </NavTab>
      </NavTabs>
    </AppBar>
  );
}
```
